Javaのオブジェクト指向プログラミング

オブジェクト指向の概要

オブジェクト指向プログラミング（OOP）は、データをオブジェクトとして扱うプログラミングパラダイムです。

クラスとオブジェクト

クラスはオブジェクトの設計図であり、オブジェクトはクラスから生成される実体です。

/* クラスとオブジェクトの例 */
public class Dog {
    private String name; // 名前
    private int age; // 年齢

    // コンストラクタ
    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // メソッド
    public void bark() {
        System.out.println(name + "が吠えました!");
    }
}

// オブジェクトの生成
Dog myDog = new Dog("ポチ", 3);
myDog.bark();

継承

クラスは他のクラスを継承し、特性を受け継ぐことができます。

/* 継承の例 */
public class Animal {
    public void eat() {
        System.out.println("動物が食べています");
    }
}

public class Cat extends Animal {
    public void meow() {
        System.out.println("猫が鳴いています");
    }
}

// 使用例
Cat myCat = new Cat();
myCat.eat(); // Animalのメソッドを使用
myCat.meow();

ポリモーフィズム

同じメソッド名で異なる動作を持つことができる能力です。

/* ポリモーフィズムの例 */
public class Vehicle {
    public void start() {
        System.out.println("車がスタートします");
    }
}

public class Bike extends Vehicle {
    @Override
    public void start() {
        System.out.println("バイクがスタートします");
    }
}

// 使用例
Vehicle myVehicle = new Bike();
myVehicle.start(); // バイクのスタートメソッドが呼び出される

カプセル化

データを隠蔽し、外部からのアクセスを制限することです。

/* カプセル化の例 */
public class BankAccount {
    private double balance; // 残高

    // メソッド
    public void deposit(double amount) {
        balance += amount;
    }

    public double getBalance() {
        return balance;
    }
}

// 使用例
BankAccount account = new BankAccount();
account.deposit(1000);
System.out.println("残高: " + account.getBalance());

まとめ

オブジェクト指向プログラミングを理解することで、効率的なコードの設計が可能になります。
