Javaのメソッド

メソッドの概要

メソッドは、特定の処理を実行するための再利用可能なコードのブロックです。

メソッドの定義

メソッドを定義する基本的な方法です。

/* メソッドの定義 */
public static void greet() {
    System.out.println("こんにちは");
}

メソッドの呼び出し

定義したメソッドを呼び出す方法です。

/* メソッドの呼び出し */
greet(); // こんにちはと表示

引数の使用

メソッドに引数を渡して処理を行います。

/* 引数を持つメソッド */
public static void greetUser(String name) {
    System.out.println("こんにちは、" + name);
}

引数を使った呼び出し

引数を使ってメソッドを呼び出します。

/* 引数を使ったメソッドの呼び出し */
greetUser("太郎"); // こんにちは、太郎と表示

戻り値の使用

メソッドから値を返す方法です。

/* 戻り値を持つメソッド */
public static int add(int a, int b) {
    return a + b;
}

戻り値を使った呼び出し

戻り値を受け取って処理を行います。

/* 戻り値を使ったメソッドの呼び出し */
int sum = add(5, 3); // sumには8が格納される
System.out.println("合計: " + sum); // 合計: 8と表示

メソッドのオーバーロード

同じ名前のメソッドを異なる引数で定義する方法です。

/* メソッドのオーバーロード */
public static int add(int a, int b) {
    return a + b;
}

public static double add(double a, double b) {
    return a + b;
}

まとめ

Javaのメソッドを理解することで、より効果的なプログラミングが可能になります。
