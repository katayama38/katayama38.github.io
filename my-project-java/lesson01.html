Javaの理解

Javaの定義

Javaは、オブジェクト指向プログラミング言語です。

/* Hello Worldのプログラム */
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

変数の宣言

Javaで変数を宣言する方法です。

/* 変数の宣言 */
int number = 10;
String text = "こんにちは";

条件分岐

条件に応じて処理を分岐する方法です。

/* 条件分岐 */
if (number > 0) {
    System.out.println("正の数です");
} else {
    System.out.println("0または負の数です");
}

ループ処理

繰り返し処理を行う方法です。

/* ループ処理 */
for (int i = 0; i < 5; i++) {
    System.out.println("カウント: " + i);
}

メソッドの定義

メソッドを定義して呼び出す方法です。

/* メソッドの定義 */
public static int add(int a, int b) {
    return a + b;
}

クラスの定義

クラスを定義する方法です。

/* クラスの定義 */
public class Animal {
    String name;

    public Animal(String name) {
        this.name = name;
    }
}

まとめ

Javaを理解することで、より効果的なプログラミングが可能になります。
