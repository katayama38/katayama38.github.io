Javaのgetter/setter

getter/setterの概要

getter/setterは、オブジェクトのフィールドへのアクセスを制御するためのメソッドです。

getterの定義

フィールドの値を取得するためのメソッドの定義方法です。

/* getterの定義 */
public class Dog {
    private String name; // 名前
    private int age; // 年齢

    // getter
    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }
}

setterの定義

フィールドの値を設定するためのメソッドの定義方法です。

/* setterの定義 */
public void setName(String name) {
    this.name = name;
}

public void setAge(int age) {
    this.age = age;
}

オブジェクトの生成

オブジェクトを生成し、setterを使ってフィールドを設定する例です。

/* オブジェクトの生成 */
Dog myDog = new Dog();
myDog.setName("ポチ"); // 名前を設定
myDog.setAge(3); // 年齢を設定

getterの使用例

getterを使ってフィールドの値を取得する例です。

/* getterの使用例 */
System.out.println("犬の名前: " + myDog.getName()); // 犬の名前: ポチと表示
System.out.println("犬の年齢: " + myDog.getAge() + "歳"); // 犬の年齢: 3歳と表示

フィールドのカプセル化

getter/setterを使うことでフィールドのカプセル化が実現されます。

/* フィールドのカプセル化 */
public class Cat {
    private String name;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }
}

まとめ

getter/setterを理解することで、オブジェクト指向プログラミングの基本的な考え方が身につきます。
