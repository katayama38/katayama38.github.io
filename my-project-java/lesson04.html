Javaの配列

配列の概要

配列は、同じデータ型の複数の値を格納するためのデータ構造です。

配列の定義

配列を定義する基本的な方法です。

/* 配列の定義 */
int[] numbers = new int[5]; // 整数型の配列を定義

配列への値の代入

配列の各要素に値を代入する方法です。

/* 配列への値の代入 */
numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;
numbers[3] = 40;
numbers[4] = 50;

配列の値の取得

配列の特定の要素の値を取得する方法です。

/* 配列の値の取得 */
int firstNumber = numbers[0]; // firstNumberには10が格納される

配列のループ処理

配列の全要素に対してループ処理を行います。

/* 配列のループ処理 */
for (int i = 0; i < numbers.length; i++) {
    System.out.println("配列の値: " + numbers[i]);
}

配列の初期化

配列を定義と同時に初期化する方法です。

/* 配列の初期化 */
int[] numbers = {10, 20, 30, 40, 50}; // 初期化と定義を同時に行う

多次元配列

配列の中に配列を持つ多次元配列についての説明です。

/* 多次元配列の定義 */
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

多次元配列の要素の取得

多次元配列から特定の要素を取得する方法です。

/* 多次元配列の要素の取得 */
int value = matrix[1][2]; // valueには6が格納される

配列の長さ

配列の長さを取得する方法です。

/* 配列の長さを取得 */
int length = numbers.length; // lengthには配列の要素数が格納される

まとめ

Javaの配列を理解することで、データの管理がより効率的になります。
