Javaプログラミングの基礎

1. Javaとは？

Javaは、1995年にサン・マイクロシステムズ（現在はオラクル）によって開発されたプログラミング言語です。主な特徴は、「オブジェクト指向」、「プラットフォーム非依存」、「セキュリティ重視」などが挙げられます。

  • オブジェクト指向: コードをオブジェクト単位で構築し、再利用性が高い。
  • プラットフォーム非依存: Java仮想マシン（JVM）があれば、どのOSでも動作。
  • セキュリティ: サンドボックス内でプログラムが動作し、安全性が確保されている。

2. Javaの開発環境の構築

Javaの開発環境を整えるには、まずJava Development Kit（JDK）をインストールします。これにより、コンパイラやライブラリ、JVMなどが使用可能になります。

IDE（統合開発環境）を使うと、コード補完やデバッグ機能が提供され、開発が効率化されます。代表的なIDEには、IntelliJ IDEAやEclipseがあります。

# JDKのインストール
公式サイトからJDKをダウンロードし、インストールします。

# Eclipseのインストール
Eclipseをインストールし、プロジェクトを作成して、Javaプログラムを作成します。

3. 基本的な文法

Javaプログラムはクラスを定義してから実行します。クラスの中にメソッドを定義し、そのメソッドが実行されることになります。

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

上記のコードでは、「Hello, World!」という文字列をコンソールに出力するプログラムです。

4. データ型と変数

Javaでは、データの型を厳密に定義します。主に使われるデータ型は、整数型（int）、浮動小数点型（double）、文字型（char）、文字列型（String）、論理型（boolean）です。

int age = 30;
double height = 175.5;
char initial = 'A';
boolean isJavaFun = true;

5. 演算子

演算子は、数値の演算や比較、論理演算に使用されます。以下のような演算子があります。

  • 算術演算子: `+`, `-`, `*`, `/`, `%`
  • 比較演算子: `==`, `!=`, `<`, `>`, `<=`, `>=`
  • 論理演算子: `&&`, `||`, `!`
int a = 5;
int b = 10;
boolean result = (a < b) && (b > 5);  // true

6. 制御フロー

条件分岐や繰り返し処理を制御フローを使って実現します。

  • if文: 条件に基づいて処理を分岐します。
  • switch文: 複数の条件に基づいて処理を分岐します。
  • ループ: 繰り返し処理を実行します。代表的なものに`for`, `while`, `do-while`があります。

if (a > b) {
    System.out.println("aはbより大きい");
} else {
    System.out.println("aはbより小さい");
}

7. 配列

配列は同じ型のデータを複数格納するための構造です。配列は固定サイズで、一度宣言したサイズを変更することはできません。

int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]); // 1番目の要素を出力

8. メソッド

メソッドは、特定の処理を実行するために定義された関数のようなものです。メソッドには引数や戻り値があります。

public static int add(int a, int b) {
    return a + b;
}

上記の`add`メソッドは、2つの整数を受け取ってその合計を返します。

9. オブジェクト指向プログラミング

Javaはオブジェクト指向プログラミング（OOP）を採用しており、データとその処理をまとめてクラスとして設計します。クラスから作成された実体をオブジェクトと呼びます。

class Person {
    String name;
    int age;
    
    public void greet() {
        System.out.println("こんにちは、" + name + "です。");
    }
}

public class Main {
    public static void main(String[] args) {
        Person p = new Person();
        p.name = "太郎";
        p.age = 25;
        p.greet();  // "こんにちは、太郎です。"と出力
    }
}

10. 継承とポリモーフィズム

継承を使うことで、既存のクラスを基に新しいクラスを作ることができます。ポリモーフィズムにより、同じメソッド名でも異なる動作をすることが可能です。

class Animal {
    public void speak() {
        System.out.println("動物が鳴きます");
    }
}

class Dog extends Animal {
    public void speak() {
        System.out.println("犬が鳴きます");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal myDog = new Dog();
        myDog.speak();  // "犬が鳴きます" と出力
    }
}

11. インターフェース

インターフェースは、クラスが実装すべきメソッドのシグネチャを定義したものです。インターフェースを使うことで、異なるクラス間で同じメソッド名を統一することができます。

interface Animal {
    void speak();
}

class Dog implements Animal {
    public void speak() {
        System.out.println("犬が鳴きます");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal myDog = new Dog();
        myDog.speak();  // "犬が鳴きます" と出力
    }
}

インターフェースを使用することで、複数の異なるクラスが同じメソッドを持つことが保証され、コードの柔軟性と拡張性が向上します。

まとめ

Javaは、オブジェクト指向に基づく堅牢なプログラミング言語であり、開発者が効率的にプログラムを作成できるツールと環境を提供しています。ここで紹介した基本的な文法や概念を理解し、さらに応用的な技術に進んでいくことで、より高度なアプリケーション開発が可能になります。
