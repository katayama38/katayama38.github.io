Javaプログラミングの詳細解説

12. 例外処理

例外処理は、プログラムの実行中に予期しない事態が発生したときに、その処理を安全に行うための方法です。`try-catch`文を使用して例外を処理します。


try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("エラー: ゼロで割ることはできません");
}

上記のコードでは、ゼロで割る操作を試みて例外が発生した場合に、`catch`ブロックでエラーメッセージを表示します。

13. コレクションフレームワーク

コレクションフレームワークは、データの集合を管理するためのインターフェースとクラスを提供します。代表的なコレクションには、`List`, `Set`, `Map`などがあります。


import java.util.*;

List list = new ArrayList<>();
list.add("Java");
list.add("Python");
list.add("C++");

for (String language : list) {
    System.out.println(language);
}

このコードでは、`ArrayList`を使って文字列のリストを作成し、ループでその内容を出力しています。

14. Javaの標準ライブラリ

Javaは多くの標準ライブラリを提供しており、これにより様々な機能を簡単に実装できます。たとえば、ファイル操作や文字列処理、日付の操作などです。


import java.util.Date;

public class Main {
    public static void main(String[] args) {
        Date date = new Date();
        System.out.println(date.toString());
    }
}

このコードでは、`Date`クラスを使って現在の日付と時刻を表示しています。

15. ファイル入出力

Javaでは、`java.io`パッケージを使ってファイルの読み書きができます。


import java.io.*;

public class FileRead {
    public static void main(String[] args) {
        try {
            BufferedReader reader = new BufferedReader(new FileReader("example.txt"));
            String line = reader.readLine();
            while (line != null) {
                System.out.println(line);
                line = reader.readLine();
            }
            reader.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

上記のコードでは、`BufferedReader`を使用してファイルを読み込む方法を示しています。

16. スレッドと並行処理

スレッドを使うことで、複数のタスクを同時に実行できます。Javaでは`Thread`クラスまたは`Runnable`インターフェースを使ってスレッドを作成します。


class MyThread extends Thread {
    public void run() {
        System.out.println("スレッドが実行されています");
    }
}

public class Main {
    public static void main(String[] args) {
        MyThread thread = new MyThread();
        thread.start();
    }
}

このコードでは、新しいスレッドを作成し、`run`メソッド内で並行処理を行っています。

17. Java 8の新機能

Java 8では、ラムダ式やStream APIなどの新機能が追加され、より簡潔で効率的なコードが書けるようになりました。


import java.util.Arrays;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List list = Arrays.asList("Java", "Python", "C++");
        list.forEach(language -> System.out.println(language));
    }
}

上記のコードでは、ラムダ式を使用してリスト内の要素を簡潔に出力しています。

18. デザインパターンの基本

デザインパターンは、ソフトウェア開発における問題解決のベストプラクティスを提供します。代表的なパターンには、SingletonやFactoryパターンがあります。


public class Singleton {
    private static Singleton instance;

    private Singleton() {}

    public static Singleton getInstance() {
        if (instance == null) {
            instance = new Singleton();
        }
        return instance;
    }
}

このコードでは、Singletonパターンを実装し、インスタンスを1つだけ作成しています。

19. JUnitによるテスト

JUnitは、Javaでユニットテストを行うためのフレームワークです。テストクラスを作成し、メソッドが期待通りに動作するかを検証します。


import org.junit.Test;
import static org.junit.Assert.*;

public class MyTest {
    @Test
    public void testAddition() {
        int result = 2 + 3;
        assertEquals(5, result);
    }
}

上記のコードは、簡単な加算のテストケースを示しています。JUnitを使うことで、コードの正確性を確認できます。

20. Javaのベストプラクティス

コードの可読性や保守性を高めるためのベストプラクティスには、命名規則や適切なコメント、冗長なコードを避けることなどがあります。


// 良い命名規則
int userAge = 25;  // 年齢を格納

良い命名規則を守ることで、他の開発者がコードを理解しやすくなります。

21. Javadocによるドキュメンテーション

Javadocは、Javaコードにコメントを追加してドキュメントを生成するツールです。メソッドやクラスの説明を追加することで、コードの理解を助けます。


/**
 * 2つの整数を加算するメソッド。
 * @param a 加算する整数
 * @param b 加算する整数
 * @return aとbの合計
 */
public int add(int a, int b) {
    return a + b;
}

このコードは、Javadocコメントを使ってメソッドの説明を付け加えています。

22. ネットワークプログラミング

Javaでは、`java.net`パッケージを使用して、ソケット通信などのネットワークプログラミングを行うことができます。


import java.net.*;

public class Main {
    public static void main(String[] args) throws Exception {
        URL url = new URL("http://example.com");
        BufferedReader reader = new BufferedReader(new InputStreamReader(url.openStream()));
        String line;
        while ((line = reader.readLine()) != null) {
            System.out.println(line);
        }
    }
}

このコードでは、指定したURLからデータを読み込み、コンソールに出力しています。
