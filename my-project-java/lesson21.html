23. データベースとの連携

Javaでデータベースと連携するには、JDBC（Java Database Connectivity）を使います。これにより、SQLデータベースとの通信を行うことができます。


import java.sql.*;

public class DatabaseExample {
    public static void main(String[] args) {
        try (Connection conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/mydatabase", "user", "password")) {
            Statement stmt = conn.createStatement();
            ResultSet rs = stmt.executeQuery("SELECT * FROM users");

            while (rs.next()) {
                System.out.println("ユーザー名: " + rs.getString("username"));
            }
        } catch (SQLException e) {
            e.printStackTrace();
        }
    }
}

上記のコードでは、MySQLデータベースに接続してユーザー情報を取得しています。

24. JavaFXによるGUI開発

JavaFXは、デスクトップアプリケーション用のGUIフレームワークです。豊富なUIコンポーネントを使って、インタラクティブなアプリケーションを作成できます。


import javafx.application.Application;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.stage.Stage;

public class SimpleApp extends Application {
    public static void main(String[] args) {
        launch(args);
    }

    @Override
    public void start(Stage primaryStage) {
        Button btn = new Button("クリックしてください");
        btn.setOnAction(e -> System.out.println("ボタンがクリックされました"));

        Scene scene = new Scene(btn, 200, 100);
        primaryStage.setTitle("JavaFX アプリケーション");
        primaryStage.setScene(scene);
        primaryStage.show();
    }
}

このコードでは、シンプルなボタンを作成し、クリック時にメッセージを表示します。

25. RESTful APIの構築

Javaでは、Spring Frameworkを使用してRESTfulなWebサービスを構築することができます。Spring Bootを使えば、手軽にAPIを作成できます。


import org.springframework.boot.*;
import org.springframework.boot.autoconfigure.*;
import org.springframework.web.bind.annotation.*;

@SpringBootApplication
@RestController
public class ApiApplication {

    @RequestMapping("/hello")
    public String hello() {
        return "Hello, World!";
    }

    public static void main(String[] args) {
        SpringApplication.run(ApiApplication.class, args);
    }
}

このコードでは、Spring Bootを使ってシンプルなRESTful APIを作成し、`/hello`エンドポイントにアクセスすると"Hello, World!"が返されます。

26. セキュリティの基本

Javaアプリケーションのセキュリティ対策として、認証や暗号化を行うことが重要です。Javaでは、`javax.crypto`パッケージを使って暗号化を行うことができます。


import javax.crypto.Cipher;
import javax.crypto.KeyGenerator;
import javax.crypto.SecretKey;

public class EncryptionExample {
    public static void main(String[] args) throws Exception {
        KeyGenerator keyGenerator = KeyGenerator.getInstance("AES");
        SecretKey secretKey = keyGenerator.generateKey();

        Cipher cipher = Cipher.getInstance("AES");
        cipher.init(Cipher.ENCRYPT_MODE, secretKey);

        String originalText = "Hello, secure world!";
        byte[] encryptedText = cipher.doFinal(originalText.getBytes());

        System.out.println("暗号化されたテキスト: " + new String(encryptedText));
    }
}

このコードでは、AESアルゴリズムを使用してテキストを暗号化しています。

27. バージョン管理とGitの活用

Gitは、ソースコードの管理に使われる分散型バージョン管理ツールです。Gitを使うことで、ソースコードの変更履歴を管理し、チームで効率的に開発ができます。


# 新しいリポジトリの作成
git init

# ファイルをステージング
git add .

# コミット
git commit -m "初回コミット"

# リモートリポジトリにプッシュ
git push origin main

上記のコマンドは、Gitでリポジトリを作成し、変更をコミットしてリモートにプッシュする基本的な流れを示しています。

28. アプリケーションのデプロイ

Javaアプリケーションのデプロイは、アプリケーションを実際のサーバーやクラウド環境に配置して運用することを意味します。デプロイ手法として、JARファイルやWARファイルを使用する方法があります。


# JARファイルを実行
java -jar myapplication.jar

このコマンドは、JARファイルを実行するためのものです。Javaアプリケーションのデプロイ後、サーバーで実行できます。

29. パフォーマンスの最適化

Javaアプリケーションのパフォーマンスを最適化するためには、メモリ管理、データ構造の選択、アルゴリズムの最適化などに注意を払う必要があります。


public class PerformanceExample {
    public static void main(String[] args) {
        long start = System.nanoTime();

        for (int i = 0; i < 1000000; i++) {
            Math.pow(i, 2);  // 何らかの計算処理
        }

        long end = System.nanoTime();
        System.out.println("処理時間: " + (end - start) + " ナノ秒");
    }
}

上記のコードは、処理時間を計測するためのサンプルコードです。パフォーマンス改善には、適切な計測と最適化が重要です。

31. リフレクションの理解

リフレクションを使うと、実行時にクラスのメタデータを操作したり、オブジェクトのプロパティを動的に取得したりすることができます。


import java.lang.reflect.Method;

public class ReflectionExample {
    public static void main(String[] args) throws Exception {
        Class clazz = Class.forName("java.util.ArrayList");
        Method method = clazz.getMethod("add", Object.class);
        System.out.println("メソッド名: " + method.getName());
    }
}

リフレクションを使うと、クラスの情報やメソッドを動的に取得できます。

32. アノテーションの利用

アノテーションは、コードにメタデータを付加するための手段です。アノテーションは、特定の処理を行うフレームワークやツールに情報を提供します。


import java.lang.annotation.*;

@Retention(RetentionPolicy.RUNTIME)
@Target(ElementType.METHOD)
public @interface MyAnnotation {
    String value() default "デフォルト値";
}

public class AnnotationExample {
    @MyAnnotation(value = "カスタム値")
    public void myMethod() {}
}

このコードでは、カスタムアノテーションを作成し、メソッドに付加しています。

33. データ構造の理解

スタックやキュー、リンクリストなどの基本的なデータ構造は、効率的なデータ処理を行うために重要です。これらの構造はアルゴリズムの設計においてよく使用されます。


// スタックの例
import java.util.Stack;

public class StackExample {
    public static void main(String[] args) {
        Stack stack = new Stack<>();
        stack.push(1);
        stack.push(2);
        stack.push(3);

        while (!stack.isEmpty()) {
            System.out.println(stack.pop());
        }
    }
}

このコードでは、スタックを使って要素を管理し、取り出す際にはLIFO（後入れ先出し）の順序で処理します。
