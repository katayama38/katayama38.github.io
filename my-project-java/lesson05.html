Javaのクラスとオブジェクト

クラスの概要

クラスは、データとメソッドをまとめて定義するための設計図です。

クラスの定義

クラスを定義する基本的な方法です。

/* クラスの定義 */
public class Dog {
    String name; // 名前
    int age; // 年齢

    // コンストラクタ
    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

オブジェクトの生成

クラスからオブジェクトを生成する方法です。

/* オブジェクトの生成 */
Dog myDog = new Dog("ポチ", 3); // 名前がポチで年齢が3の犬を作成

オブジェクトのプロパティへのアクセス

オブジェクトのプロパティにアクセスする方法です。

/* プロパティへのアクセス */
System.out.println("犬の名前: " + myDog.name); // 犬の名前: ポチと表示
System.out.println("犬の年齢: " + myDog.age + "歳"); // 犬の年齢: 3歳と表示

メソッドの定義

クラス内にメソッドを定義する方法です。

/* メソッドの定義 */
public void bark() {
    System.out.println(name + "がワンワンと鳴いた！");
}

メソッドの呼び出し

オブジェクトからメソッドを呼び出す方法です。

/* メソッドの呼び出し */
myDog.bark(); // ポチがワンワンと鳴いた！と表示

クラスの継承

クラスを継承して新しいクラスを作成する方法です。

/* クラスの継承 */
public class Puppy extends Dog {
    public Puppy(String name) {
        super(name, 0); // 子クラスのコンストラクタを呼び出す
    }
}

ポリモーフィズム

同じメソッド名で異なる実装を持つことができる特性です。

/* ポリモーフィズムの例 */
@Override
public void bark() {
    System.out.println(name + "が小さくワンと鳴いた！");
}

まとめ

Javaのクラスとオブジェクトを理解することで、より複雑なプログラムを設計できます。
