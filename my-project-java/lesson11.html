Javaの継承とポリモーフィズム

継承の概要

継承は、既存のクラスを基に新しいクラスを作成するためのメカニズムです。親クラスの特性を子クラスが受け継ぐことができます。

基本的な継承の例

親クラスから子クラスを作成する方法を示します。

/* 継承の基本例 */
public class Animal {
    public void eat() {
        System.out.println("動物が食べています");
    }
}

public class Dog extends Animal {
    public void bark() {
        System.out.println("犬が吠えました!");
    }
}

// 使用例
Dog myDog = new Dog();
myDog.eat(); // 親クラスのメソッドを使用
myDog.bark(); // 子クラスのメソッド

ポリモーフィズムの概要

ポリモーフィズムとは、同じメソッド名で異なる動作を持つことができる能力です。

ポリモーフィズムの例

異なるクラスが同じメソッドをオーバーライドする例です。

/* ポリモーフィズムの例 */
public class Vehicle {
    public void start() {
        System.out.println("車がスタートします");
    }
}

public class Car extends Vehicle {
    @Override
    public void start() {
        System.out.println("自動車がスタートします");
    }
}

public class Bike extends Vehicle {
    @Override
    public void start() {
        System.out.println("バイクがスタートします");
    }
}

// 使用例
Vehicle myVehicle = new Car();
myVehicle.start(); // 自動車のスタートメソッドが呼び出される

myVehicle = new Bike();
myVehicle.start(); // バイクのスタートメソッドが呼び出される

抽象クラスとインターフェース

ポリモーフィズムを実現するために抽象クラスやインターフェースを使用します。

/* 抽象クラスとインターフェースの例 */
abstract class Animal {
    abstract void sound(); // 抽象メソッド
}

class Cat extends Animal {
    void sound() {
        System.out.println("猫が鳴いています");
    }
}

class Dog extends Animal {
    void sound() {
        System.out.println("犬が吠えています");
    }
}

// 使用例
Animal myAnimal = new Cat();
myAnimal.sound(); // 猫の鳴き声

myAnimal = new Dog();
myAnimal.sound(); // 犬の吠え声

まとめ

継承とポリモーフィズムを理解することで、コードの再利用性と柔軟性が向上します。
