Javaのラムダ式

ラムダ式の概要

ラムダ式は、Java 8から導入された新しい記法で、関数型プログラミングをサポートします。簡潔に記述できるため、コードの可読性が向上します。

基本的な構文

ラムダ式は、引数リスト、矢印（->）、および式本体から構成されます。

/* ラムダ式の基本的な構文 */
(x, y) -> x + y

リストの処理におけるラムダ式の利用

リストの要素を操作する際に、ラムダ式を使用してコードを簡潔に記述できます。

/* ラムダ式を用いたリストの処理 */
import java.util.Arrays;
import java.util.List;

public class LambdaExample {
    public static void main(String[] args) {
        List fruits = Arrays.asList("リンゴ", "バナナ", "オレンジ");
        fruits.forEach(fruit -> System.out.println(fruit));
    }
}

関数型インターフェイスの利用

ラムダ式は関数型インターフェイスに対して使用されます。これにより、より抽象的なコードが書けます。

/* 関数型インターフェイスの例 */
@FunctionalInterface
interface MyFunctionalInterface {
    void execute();
}

public class FunctionalInterfaceExample {
    public static void main(String[] args) {
        MyFunctionalInterface myFunc = () -> System.out.println("関数型インターフェイスの実行");
        myFunc.execute(); // ラムダ式の実行
    }
}

ラムダ式を用いたコレクションのフィルタリング

ストリームAPIと組み合わせることで、コレクションのフィルタリングが簡単に行えます。

/* ストリームとラムダ式を用いたフィルタリング */
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class StreamLambdaExample {
    public static void main(String[] args) {
        List numbers = Arrays.asList(1, 2, 3, 4, 5, 6);
        List evenNumbers = numbers.stream()
                                            .filter(n -> n % 2 == 0)
                                            .collect(Collectors.toList());
        System.out.println("偶数: " + evenNumbers);
    }
}

まとめ

Javaのラムダ式を利用することで、コードが簡潔になり、可読性が向上します。関数型プログラミングの考え方を取り入れることで、より柔軟なプログラムを作成できます。
