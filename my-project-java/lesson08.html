Javaのパッケージとインポート

パッケージの概要

パッケージは、関連するクラスをまとめるための名前空間です。

パッケージの定義

パッケージを定義する基本的な方法です。

/* パッケージの定義 */
package com.example.animals;

public class Dog {
    private String name; // 名前
    private int age; // 年齢

    // コンストラクタ
    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

パッケージの使用

別のクラスからパッケージ内のクラスを使用する方法です。

/* パッケージの使用 */
import com.example.animals.Dog;

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog("ポチ", 3); // Dogクラスを使用
        System.out.println("犬の名前: " + myDog.name);
    }
}

デフォルトパッケージ

パッケージを指定しない場合、デフォルトパッケージに配置されます。

/* デフォルトパッケージ */
public class Cat {
    private String name;

    public Cat(String name) {
        this.name = name;
    }
}

インポート文の使い方

特定のクラスやすべてのクラスをインポートする方法です。

/* インポート文の使用 */
import com.example.animals.*; // すべてのクラスをインポート

パッケージの重要性

パッケージを使うことで、クラスの整理や再利用性が向上します。

/* パッケージの重要性 */
package com.example.utils;

public class MathUtil {
    public static int add(int a, int b) {
        return a + b;
    }
}

まとめ

パッケージとインポートを理解することで、Javaプログラムの構造が明確になります。
