Javaのコンストラクタ

コンストラクタの概要

コンストラクタは、オブジェクトが生成される際に初期化を行う特別なメソッドです。

コンストラクタの定義

コンストラクタを定義する基本的な方法です。

/* コンストラクタの定義 */
public class Dog {
    String name; // 名前
    int age; // 年齢

    // コンストラクタ
    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

オブジェクトの生成

コンストラクタを使ってオブジェクトを生成する方法です。

/* オブジェクトの生成 */
Dog myDog = new Dog("ポチ", 3); // 名前がポチで年齢が3の犬を作成

デフォルトコンストラクタ

引数を持たないコンストラクタの例です。

/* デフォルトコンストラクタ */
public Dog() {
    this.name = "無名";
    this.age = 0;
}

コンストラクタのオーバーロード

同じクラス内で異なる引数を持つ複数のコンストラクタを定義することです。

/* コンストラクタのオーバーロード */
public Dog(String name) {
    this.name = name;
    this.age = 0; // 年齢はデフォルトで0
}

コンストラクタの使用例

コンストラクタを使ってオブジェクトを初期化する例です。

/* コンストラクタの使用例 */
Dog myDog = new Dog("ポチ", 3);
System.out.println("犬の名前: " + myDog.name); // 犬の名前: ポチと表示
System.out.println("犬の年齢: " + myDog.age + "歳"); // 犬の年齢: 3歳と表示

クラスの継承とコンストラクタ

親クラスのコンストラクタを呼び出す方法です。

/* クラスの継承とコンストラクタ */
public class Puppy extends Dog {
    public Puppy(String name) {
        super(name, 0); // 親クラスのコンストラクタを呼び出す
    }
}

まとめ

コンストラクタを理解することで、オブジェクトの初期化が効率的に行えます。
