Javaの制御フロー

制御フローの概要

制御フローは、プログラムの実行順序を制御する構文です。

条件分岐

条件に応じて処理を分岐する方法です。

/* 条件分岐 */
if (条件) {
    // 条件が真のときの処理
} else {
    // 条件が偽のときの処理
}

switch文

複数の条件を扱う際に使います。

/* switch文 */
switch (変数) {
    case 値1:
        // 値1の場合の処理
        break;
    case 値2:
        // 値2の場合の処理
        break;
    default:
        // どのcaseにも該当しない場合の処理
}

ループ処理

繰り返し処理を行う方法です。

/* forループ */
for (int i = 0; i < 5; i++) {
    System.out.println("カウント: " + i);
}

/* whileループ */
int i = 0;
while (i < 5) {
    System.out.println("カウント: " + i);
    i++;
}

do-whileループ

条件が真である限り繰り返し処理を行います。

/* do-whileループ */
int j = 0;
do {
    System.out.println("カウント: " + j);
    j++;
} while (j < 5);

break文とcontinue文

ループ内の処理を制御する方法です。

/* break文 */
for (int k = 0; k < 10; k++) {
    if (k == 5) {
        break; // kが5のときループを終了
    }
}

/* continue文 */
for (int k = 0; k < 10; k++) {
    if (k % 2 == 0) {
        continue; // 偶数のときは次のループへ
    }
    System.out.println(k); // 奇数のみ表示
}

例外処理

プログラムの実行中に発生するエラーを処理します。

/* 例外処理 */
try {
    int result = 10 / 0; // ゼロ除算を試みる
} catch (ArithmeticException e) {
    System.out.println("エラー: ゼロで割ることはできません");
}

まとめ

Javaの制御フローを理解することで、柔軟で効率的なプログラミングが可能になります。
