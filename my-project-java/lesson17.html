JavaのストリームAPI

ストリームAPIの概要

ストリームAPIは、Java 8で導入された機能で、コレクションの操作を簡潔かつ効率的に行うための手段を提供します。

基本的な使い方

ストリームは、コレクションの要素に対して連続的な操作を行うための抽象化です。

/* ストリームの基本的な使い方 */
import java.util.Arrays;
import java.util.List;

public class StreamBasicExample {
    public static void main(String[] args) {
        List fruits = Arrays.asList("リンゴ", "バナナ", "オレンジ");
        fruits.stream().forEach(System.out::println); // 各要素を出力
    }
}

フィルタリングの例

ストリームAPIを使用して、特定の条件に基づいて要素をフィルタリングできます。

/* ストリームを用いたフィルタリング */
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class StreamFilterExample {
    public static void main(String[] args) {
        List numbers = Arrays.asList(1, 2, 3, 4, 5, 6);
        List evenNumbers = numbers.stream()
                                            .filter(n -> n % 2 == 0)
                                            .collect(Collectors.toList());
        System.out.println("偶数: " + evenNumbers);
    }
}

マッピングの例

ストリームAPIを使用して、要素を別の形式に変換することができます。

/* ストリームを用いたマッピング */
import java.util.Arrays;
import java.util.List;
import java.util.stream.Collectors;

public class StreamMapExample {
    public static void main(String[] args) {
        List fruits = Arrays.asList("リンゴ", "バナナ", "オレンジ");
        List upperFruits = fruits.stream()
                                          .map(String::toUpperCase)
                                          .collect(Collectors.toList());
        System.out.println("大文字の果物: " + upperFruits);
    }
}

集約操作の例

ストリームAPIを使用して、コレクションの集約操作を行うことができます。

/* ストリームを用いた集約操作 */
import java.util.Arrays;
import java.util.List;

public class StreamReduceExample {
    public static void main(String[] args) {
        List numbers = Arrays.asList(1, 2, 3, 4, 5);
        int sum = numbers.stream()
                         .reduce(0, Integer::sum);
        System.out.println("合計: " + sum);
    }
}

まとめ

JavaのストリームAPIを活用することで、コレクションの操作を簡潔に行い、コードの可読性を向上させることができます。
