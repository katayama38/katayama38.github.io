jQueryの基礎と応用

画像の遅延読み込み

ウェブページのパフォーマンス向上のために、画像を必要な時に読み込む「遅延読み込み」を実装します。jQueryを使って、画像が表示される直前に読み込むようにします。

$(document).ready(function() {
    $("img.lazy").each(function() {
        var img = $(this);
        img.attr("src", img.data("src"));
        img.removeClass("lazy");
    });
});

カスタムアニメーションの作成

jQueryの`animate`メソッドを使って、独自のアニメーションを作成します。例えば、要素を移動させる、サイズを変更する、色を変更するなど、アニメーションの幅を広げます。

$(document).ready(function() {
    $("#animateBtn").click(function() {
        $(".box").animate({
            width: "300px",
            height: "300px",
            left: "200px"
        }, 1000);
    });
});

外部APIの利用

jQueryで外部APIを呼び出し、そのデータをウェブページに表示します。以下の例では、JSON形式のデータを取得して表示しています。

$(document).ready(function() {
    $.ajax({
        url: "https://jsonplaceholder.typicode.com/posts",
        method: "GET",
        dataType: "json",
        success: function(response) {
            $.each(response, function(index, post) {
                $("#posts").append("
" + post.title + "
");
            });
        }
    });
});

テーマの切り替え

ユーザーがテーマ（例：ダークモードとライトモード）を切り替えられる機能をjQueryで実装します。以下のコードでは、ボタンをクリックしてテーマを変更します。

$(document).ready(function() {
    $("#themeToggle").click(function() {
        $("body").toggleClass("dark-theme");
    });
});

多言語対応の実装

jQueryを使って、簡易的な多言語対応機能を作成します。ユーザーが選択した言語に応じて、テキストを切り替える方法です。

$(document).ready(function() {
    $("#langSelect").change(function() {
        var lang = $(this).val();
        $(".lang").each(function() {
            if ($(this).data("lang") === lang) {
                $(this).show();
            } else {
                $(this).hide();
            }
        });
    });
});

印刷用スタイルの適用

ページの印刷時に適用されるスタイルをjQueryで変更する方法です。印刷プレビューをカスタマイズして、不要な要素を非表示にできます。

$(document).ready(function() {
    $(window).on("beforeprint", function() {
        $(".no-print").hide();
    });
    $(window).on("afterprint", function() {
        $(".no-print").show();
    });
});

グラフの描画

jQueryと他のライブラリ（例えばChart.js）を使って、データをグラフとして視覚化します。以下は基本的なグラフの描画例です。

$(document).ready(function() {
    var ctx = document.getElementById("myChart").getContext("2d");
    var myChart = new Chart(ctx, {
        type: "bar",
        data: {
            labels: ["Red", "Blue", "Yellow", "Green", "Purple", "Orange"],
            datasets: [{
                label: "My First Dataset",
                data: [12, 19, 3, 5, 2, 3],
                backgroundColor: [
                    'rgba(255, 99, 132, 0.2)',
                    'rgba(54, 162, 235, 0.2)',
                    'rgba(255, 206, 86, 0.2)',
                    'rgba(75, 192, 192, 0.2)',
                    'rgba(153, 102, 255, 0.2)',
                    'rgba(255, 159, 64, 0.2)'
                ],
                borderColor: [
                    'rgba(255, 99, 132, 1)',
                    'rgba(54, 162, 235, 1)',
                    'rgba(255, 206, 86, 1)',
                    'rgba(75, 192, 192, 1)',
                    'rgba(153, 102, 255, 1)',
                    'rgba(255, 159, 64, 1)'
                ],
                borderWidth: 1
            }]
        },
        options: {
            scales: {
                y: {
                    beginAtZero: true
                }
            }
        }
    });
});

ユーザー入力のトラッキング

フォームや入力フィールドの変更をリアルタイムで監視し、ユーザーの入力内容をトラッキングする方法です。

$(document).ready(function() {
    $("input, textarea").on("input", function() {
        var inputValue = $(this).val();
        $("#trackOutput").text(inputValue);
    });
});

モバイルデバイス向けの最適化

モバイルデバイスに最適なUIを作成するために、画面サイズを動的に判断して、レイアウトや要素を変更します。

$(document).ready(function() {
    if ($(window).width() < 768) {
        $(".menu").hide();
    } else {
        $(".menu").show();
    }

    $(window).resize(function() {
        if ($(window).width() < 768) {
            $(".menu").hide();
        } else {
            $(".menu").show();
        }
    });
});

    • 外部ライブラリとの統合

    jQueryと他のJavaScriptライブラリを連携させる方法です。例えば、`moment.js`を使って日付操作を簡単に行います。

    $(document).ready(function() {
    var date = moment().format("YYYY-MM-DD");
    $("#dateOutput").text(date);
});

    SEOへの配慮

    jQueryを使用する際に、SEOに配慮するためのポイントについて説明します。例えば、動的に読み込まれるコンテンツが検索エンジンに影響を与えないように工夫します。

    $(document).ready(function() {
    if ($("#dynamicContent").length) {
        console.log("SEO対応済み");
    }
});

    テストの自動化

    jQueryを使った簡単なテストケースの自動化を行います。`QUnit`などのテスティングフレームワークと組み合わせて使用できます。

    $(document).ready(function() {
    $("#testButton").click(function() {
        var result = $("#inputField").val();
        if (result === "expected") {
            alert("テスト成功");
        } else {
            alert("テスト失敗");
        }
    });
});
