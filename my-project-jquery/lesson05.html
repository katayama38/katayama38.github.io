jQueryの基礎と応用

外部リソースの読み込み

Ajaxを使用して、他のHTMLファイルや画像を動的に読み込む方法を紹介します。これにより、ページ全体をリロードせずに新しいコンテンツを追加できます。

$(document).ready(function() {
    $("#loadContent").click(function() {
        $("#content").load("external-content.html");
    });
});

セレクタの最適化

効率的なセレクタの使い方について学び、パフォーマンスを向上させる方法を紹介します。過度に広範囲なセレクタは避け、特定のセレクタを使用しましょう。

$(document).ready(function() {
    // 効率的なセレクタ
    $(".menu-item").each(function() {
        $(this).css("color", "blue");
    });

    // 非効率的なセレクタ
    $("div p").each(function() {
        $(this).css("color", "red");
    });
});

プラグインの作成

自分だけのjQueryプラグインを作成する方法を学びます。カスタムな機能を提供するために、jQueryの拡張を行います。

(function($) {
    $.fn.extend({
        highlight: function(color) {
            this.css("background-color", color);
            return this;
        }
    });
})(jQuery);

$(document).ready(function() {
    $(".highlight").highlight("yellow");
});

jQueryのイベントモデル

jQueryにおけるイベントの仕組みと、イベントを効率的に管理する方法を紹介します。イベントのデリゲーションや名前空間を活用します。

$(document).ready(function() {
    $(".button").on("click", function() {
        alert("ボタンがクリックされました");
    });

    // イベントのデリゲーション
    $("#parent").on("click", ".child", function() {
        alert("子要素がクリックされました");
    });
});

フォームの自動入力

jQueryを使って、ユーザーがフォームに入力した内容を自動的に入力する方法を紹介します。

$(document).ready(function() {
    $("#autoFill").click(function() {
        $("#name").val("山田 太郎");
        $("#email").val("taro@example.com");
    });
});

ユーザーインターフェースの改善

ユーザー体験を向上させるために、jQueryでインタラクティブなUI要素を作成します。例えば、ダイアログやアニメーションを使用して、操作をより魅力的にします。

$(document).ready(function() {
    $(".open-dialog").click(function() {
        $("#dialog").fadeIn();
    });

    $(".close-dialog").click(function() {
        $("#dialog").fadeOut();
    });
});

スクロールイベントの活用

スクロール位置に応じて動作する機能を実装します。例えば、スクロールが一定位置を超えるとナビゲーションバーが固定されるなど。

$(document).ready(function() {
    $(window).scroll(function() {
        if ($(this).scrollTop() > 100) {
            $(".fixed-nav").addClass("active");
        } else {
            $(".fixed-nav").removeClass("active");
        }
    });
});

データのフィルタリング

リストやテーブルのデータをフィルタリングする方法を学びます。ユーザーが入力した条件に基づいて動的にデータを絞り込みます。

$(document).ready(function() {
    $("#filterInput").on("keyup", function() {
        var value = $(this).val().toLowerCase();
        $(".data-list li").filter(function() {
            $(this).toggle($(this).text().toLowerCase().indexOf(value) > -1);
        });
    });
});

クッキーの操作

jQueryを使って、クッキーを読み書きする方法を紹介します。ユーザーの情報を保存し、次回訪問時に利用することができます。

$(document).ready(function() {
    $("#setCookie").click(function() {
        document.cookie = "username=John Doe";
    });

    $("#getCookie").click(function() {
        alert(document.cookie);
    });
});

簡単なゲームの作成

jQueryを使用して、シンプルなゲームを作成する方法を学びます。ここでは、クリックイベントを使って簡単なゲームを実装します。

$(document).ready(function() {
    var score = 0;

    $(".game-item").click(function() {
        score++;
        $("#score").text("スコア: " + score);
    });
});

非同期処理の活用

jQueryの`$.ajax`を使用して非同期処理を行い、サーバーからデータを非同期で取得する方法を紹介します。

$(document).ready(function() {
    $("#fetchData").click(function() {
        $.ajax({
            url: "https://jsonplaceholder.typicode.com/posts",
            type: "GET",
            dataType: "json",
            success: function(data) {
                var output = "";
                $.each(data, function(index, post) {
                    output += "
" + post.title + "
";
                });
                $("#dataContainer").html(output);
            }
        });
    });
});
