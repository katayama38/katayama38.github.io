jQueryの基礎と応用

jQuery Mobileの導入

jQuery Mobileを使用して、モバイルフレンドリーなウェブアプリケーションを作成する方法を紹介します。簡単なUIコンポーネントやレスポンシブなレイアウトを実装します。

$(document).ready(function() {
    // jQuery Mobileの基本設定
    $("button").click(function() {
        $(this).addClass("ui-btn-active");
    });
});

モーダルウィンドウの作成

jQueryを使って、モーダルダイアログを簡単に作成する方法を学びます。クリックイベントをトリガーにして、ダイアログを開閉します。

$(document).ready(function() {
    $("#openModal").click(function() {
        $("#myModal").fadeIn();
    });
    $("#closeModal").click(function() {
        $("#myModal").fadeOut();
    });
});

スライダーの実装

画像やコンテンツをスライドさせるために、jQueryを使用したシンプルなスライダーの作成方法を紹介します。

$(document).ready(function() {
    var currentIndex = 0;
    var images = $(".slider img");
    
    function showSlide(index) {
        images.hide().eq(index).fadeIn();
    }

    $("#next").click(function() {
        currentIndex = (currentIndex + 1) % images.length;
        showSlide(currentIndex);
    });

    $("#prev").click(function() {
        currentIndex = (currentIndex - 1 + images.length) % images.length;
        showSlide(currentIndex);
    });

    showSlide(currentIndex);
});

タブの実装

jQueryを使ってタブ形式のコンテンツ切り替えを実装します。タブのクリックで異なるコンテンツを表示します。

$(document).ready(function() {
    $(".tabs li").click(function() {
        var tab_id = $(this).attr("data-tab");
        $(".tabs li").removeClass("active");
        $(".tab-content").removeClass("active");
        $(this).addClass("active");
        $("#" + tab_id).addClass("active");
    });
});

フォームのバリデーション

フォーム入力時にリアルタイムでバリデーションを行い、不正なデータの送信を防ぐ方法を紹介します。

$(document).ready(function() {
    $("#submit").click(function(event) {
        var email = $("#email").val();
        var regex = /^[\w-]+(\.[\w-]+)*@[\w-]+(\.[\w-]+)+$/;
        if (!regex.test(email)) {
            alert("無効なメールアドレスです");
            event.preventDefault();
        }
    });
});

レスポンシブデザインのサポート

メディアクエリとjQueryを組み合わせて、画面サイズに応じたレイアウトの変更や要素の表示・非表示を実装します。

$(document).ready(function() {
    if ($(window).width() < 768) {
        $(".menu").hide();
    } else {
        $(".menu").show();
    }

    $(window).resize(function() {
        if ($(window).width() < 768) {
            $(".menu").hide();
        } else {
            $(".menu").show();
        }
    });
});

ホバー効果の実装

マウスオーバー時にアニメーションやエフェクトを追加して、インタラクティブな体験を作成する方法を紹介します。

$(document).ready(function() {
    $(".hover-item").hover(function() {
        $(this).fadeTo(200, 0.7);
    }, function() {
        $(this).fadeTo(200, 1);
    });
});

アコーディオンメニューの作成

クリックすることでコンテンツがスライドして表示されるアコーディオンスタイルのメニューをjQueryで実装します。

$(document).ready(function() {
    $(".accordion-header").click(function() {
        $(this).next(".accordion-content").slideToggle();
    });
});

ページスムーススクロール

ページ内のリンクをクリックした際に、スムーズにスクロールする方法をjQueryで実装します。

$(document).ready(function() {
    $("a[href^='#']").click(function(event) {
        var target = $($(this).attr("href"));
        $("html, body").animate({ scrollTop: target.offset().top }, 1000);
        event.preventDefault();
    });
});

カスタムイベントの作成

独自のカスタムイベントをjQueryで定義し、特定のタイミングでトリガーする方法を学びます。

$(document).ready(function() {
    $("#triggerEvent").click(function() {
        $(document).trigger("customEvent");
    });

    $(document).on("customEvent", function() {
        alert("カスタムイベントが発火しました！");
    });
});

Ajaxのデータ受信

Ajaxを使ってサーバーからJSONデータを受信し、それをHTMLに表示する方法を学びます。

$(document).ready(function() {
    $.ajax({
        url: "api/data.json",
        method: "GET",
        dataType: "json",
        success: function(response) {
            $("#result").html("受信データ: " + response.data);
        }
    });
});
