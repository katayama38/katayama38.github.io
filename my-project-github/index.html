私について

16歳で社会へ飛び込み仕事を通して人と社会の関わり方を多く学びました。自身の性格と過去の経験値を存分に発揮できる環境が、Webだと思いました。エラーしか表示されない時や自身のイメージ通りになった時。問題解決やエラー解決の模索などが楽しく、自分が好きな事なので学びを続けております。

私とWeb

私自身、過去の職場でパソコンに触れる事はありました。Excel、Word、動画編集、曲編集など。ですが、Web業は未経験なので年齢を気にせず、フランクに厳しく接していただけると幸いです。出来る事の増加と進化の発展速度の速い業界なので、とにかく学び続けます。楽しいので。
