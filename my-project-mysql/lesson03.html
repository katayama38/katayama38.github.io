MySQL基本的なSQL文の理解

SELECT文

SELECT文は、データベースからデータを取得するために使用します。

/* データの取得 */
SELECT * FROM users;

INSERT文

INSERT文は、新しいデータをテーブルに追加するために使用します。

/* データの挿入 */
INSERT INTO users (name, email) VALUES ('山田太郎', 'yamada@example.com');

UPDATE文

UPDATE文は、既存のデータを更新するために使用します。

/* データの更新 */
UPDATE users SET email = 'new_email@example.com' WHERE id = 1;

DELETE文

DELETE文は、不要なデータをテーブルから削除するために使用します。

/* データの削除 */
DELETE FROM users WHERE id = 1;

CREATE TABLE文

CREATE TABLE文は、新しいテーブルをデータベースに作成するために使用します。

/* テーブルの作成 */
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100)
);

ALTER TABLE文

ALTER TABLE文は、既存のテーブルの構造を変更するために使用します。

/* テーブルの変更 */
ALTER TABLE users ADD COLUMN age INT;

まとめ

基本的なSQL文を理解することで、データベース操作が容易になります。
