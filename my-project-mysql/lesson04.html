MySQLテーブルの作成と管理の理解

CREATE TABLE文

CREATE TABLE文は、新しいテーブルをデータベースに作成するために使用します。

/* テーブルの作成 */
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100),
    email VARCHAR(100)
);

ALTER TABLE文

ALTER TABLE文は、既存のテーブルの構造を変更するために使用します。

/* テーブルの変更 */
ALTER TABLE users ADD COLUMN age INT;

DROP TABLE文

DROP TABLE文は、テーブルをデータベースから削除するために使用します。

/* テーブルの削除 */
DROP TABLE users;

SHOW TABLES文

SHOW TABLES文は、データベース内のすべてのテーブルを表示します。

/* テーブルの表示 */
SHOW TABLES;

DESCRIBE文

DESCRIBE文は、特定のテーブルの構造を表示します。

/* テーブルの構造表示 */
DESCRIBE users;

CREATE INDEX文

CREATE INDEX文は、テーブルの特定の列にインデックスを作成します。

/* インデックスの作成 */
CREATE INDEX idx_email ON users (email);

まとめ

テーブルの作成と管理を理解することで、データベースの運用がスムーズになります。
