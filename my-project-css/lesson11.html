CSSフォントとタイポグラフィの理解

フォントとは

フォントは、テキストの見た目を決定する重要な要素です。

/* フォントの基本例 */
body {
    font-family: 'Arial', sans-serif; /* フォントファミリーを指定 */
}

タイポグラフィの重要性

タイポグラフィは、情報の伝達において視覚的な効果を高める役割を果たします。

/* タイポグラフィの例 */
h1 {
    font-size: 2.5em; /* 見出しのサイズを指定 */
    font-weight: bold; /* フォントの太さを指定 */
}

p {
    line-height: 1.6; /* 行間を設定 */
}

フォントサイズの調整

フォントサイズを調整することで、テキストの可読性を向上させることができます。

/* フォントサイズの設定 */
:root {
    --base-font-size: 16px; /* 基本フォントサイズ */
}

body {
    font-size: var(--base-font-size); /* 基本フォントサイズを使用 */
}

h2 {
    font-size: calc(var(--base-font-size) * 1.5); /* h2のフォントサイズを計算 */
}

フォントのスタイル

フォントのスタイルを変更することで、デザインに変化を与えられます。

/* フォントスタイルの例 */
.italic {
    font-style: italic; /* イタリック体 */
}

.underline {
    text-decoration: underline; /* 下線を引く */
}

フォントの色と背景

フォントの色と背景色を組み合わせることで、視覚的なコントラストを高めることができます。

/* フォントの色と背景色 */
.text-highlight {
    color: white; /* テキストの色 */
    background-color: #3498db; /* 背景色 */
}

レスポンシブタイポグラフィ

レスポンシブデザインでは、画面サイズに応じてフォントサイズを変更することが重要です。

/* レスポンシブフォントサイズの例 */
@media (max-width: 600px) {
    body {
        font-size: 14px; /* 小さい画面用のフォントサイズ */
    }
}

@media (min-width: 601px) {
    body {
        font-size: 16px; /* 大きい画面用のフォントサイズ */
    }
}

まとめ

フォントとタイポグラフィを適切に使用することで、ユーザーにとって読みやすく、魅力的なウェブデザインを作成できます。
