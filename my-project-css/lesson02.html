CSSセレクタの理解

CSSの基本構造

CSS（Cascading Style Sheets）は、HTML文書の見た目を制御するためのスタイルシート言語です。スタイルは、外部スタイルシートや内部スタイルシート、インラインスタイルとして適用できます。

/* 外部スタイルシートの例 */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

要素セレクタ

要素セレクタは、HTMLの特定の要素を選択します。例えば、全ての`

`タグにスタイルを適用することができます。

/* 要素セレクタの例 */
p {
    color: blue;
    font-size: 16px;
}

クラスセレクタ

クラスセレクタは、特定のクラス名を持つ要素にスタイルを適用します。クラスは複数の要素に使い回すことができます。

/* クラスセレクタの例 */
.highlight {
    color: red;
    background-color: yellow;
}

IDセレクタ

IDセレクタは、特定のIDを持つ要素にのみスタイルを適用します。IDは文書内で一意でなければなりません。

/* IDセレクタの例 */
#main-title {
    font-size: 24px;
    font-weight: bold;
}

属性セレクタ

属性セレクタは、特定の属性を持つ要素を選択します。例えば、特定の`type`属性を持つ``タグを選ぶことができます。

/* 属性セレクタの例 */
input[type="text"] {
    border: 1px solid #ccc;
    padding: 5px;
}

擬似クラス

擬似クラスは、要素の特定の状態に基づいてスタイルを適用します。例えば、マウスオーバー時のスタイルなどがあります。

/* 擬似クラスの例 */
a:hover {
    color: green;
    text-decoration: underline;
}

まとめ

CSSセレクタを理解することで、スタイルを効率的に適用し、ウェブページのデザインを向上させることができます。
