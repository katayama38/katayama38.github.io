CSSの基礎の理解

CSSの基本構造

CSS（Cascading Style Sheets）は、HTML文書の見た目を制御するためのスタイルシート言語です。スタイルは、外部スタイルシートや内部スタイルシート、インラインスタイルとして適用できます。

/* 外部スタイルシートの例 */
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

CSSセレクタの種類

CSSセレクタは、スタイルを適用する要素を選択するために使用します。セレクタには、要素セレクタ、クラスセレクタ、IDセレクタ、属性セレクタなどがあります。

/* クラスセレクタの例 */
.highlight {
    color: red;
}

/* IDセレクタの例 */
#main-title {
    font-size: 24px;
}

ボックスモデル

ボックスモデルは、すべてのHTML要素がボックスとして扱われることを説明します。ボックスは、マージン、ボーダー、パディング、コンテンツで構成されています。

.box {
    width: 100px;
    height: 100px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}

レスポンシブデザイン

レスポンシブデザインは、異なるデバイスや画面サイズに適応するウェブデザインの手法です。CSSメディアクエリを使用して、スタイルを変更できます。

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

CSSフレームワーク

CSSフレームワークは、スタイルの迅速な開発を支援するためのライブラリです。BootstrapやTailwind CSSなど、さまざまなフレームワークがあります。

まとめ

CSSを理解することで、ウェブページの見た目を向上させ、ユーザーにより良い体験を提供できます。
