CSSアニメーションとトランジションの理解

アニメーションとは

アニメーションは、CSSを使用して要素の状態を変化させ、視覚的な動きを作り出す手法です。

/* アニメーションの基本例 */
@keyframes example {
    from {transform: translateX(0);} /* 初期位置 */
    to {transform: translateX(100px);} /* 移動後の位置 */
}

.box {
    width: 100px;
    height: 100px;
    background-color: blue;
    animation: example 2s; /* アニメーションを適用 */
}

トランジションとは

トランジションは、要素のプロパティの変化を滑らかにするための機能です。

/* トランジションの基本例 */
.box {
    width: 100px;
    height: 100px;
    background-color: blue;
    transition: background-color 0.5s; /* 背景色の変化にトランジションを適用 */
}

.box:hover {
    background-color: red; /* マウスオーバー時に背景色を変更 */
}

アニメーションの詳細設定

アニメーションには、持続時間、タイミング関数、遅延時間などの詳細設定が可能です。

/* アニメーションの詳細設定 */
.box {
    animation: example 2s ease-in-out 1s infinite; /* イージングと遅延、繰り返しを設定 */
}

トランジションのプロパティ指定

トランジションを特定のプロパティに適用することもできます。

/* 特定のプロパティにトランジションを適用 */
.box {
    transition: width 0.5s, height 0.5s; /* 幅と高さの変化にトランジションを設定 */
}

.box:hover {
    width: 150px; /* マウスオーバー時に幅を変更 */
    height: 150px; /* マウスオーバー時に高さを変更 */
}

アニメーションの遅延と繰り返し

アニメーションには、遅延時間や繰り返しの設定が可能です。

/* アニメーションの遅延と繰り返し */
@keyframes move {
    from {transform: translateY(0);}
    to {transform: translateY(100px);}
}

.box {
    animation: move 1s ease-in-out 0.5s infinite alternate; /* 遅延と繰り返しを設定 */
}

アニメーションの中断

アニメーションを中断するには、animation-play-stateプロパティを使用します。

/* アニメーションの中断 */
.box {
    animation: example 2s;
    animation-play-state: paused; /* アニメーションを一時停止 */
}

.box:hover {
    animation-play-state: running; /* マウスオーバー時にアニメーションを再開 */
}

まとめ

アニメーションとトランジションを理解することで、ウェブページに動きを持たせ、ユーザー体験を向上させることができます。
