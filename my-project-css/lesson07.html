CSS擬似クラスと擬似要素の理解

擬似クラスとは

擬似クラスは、特定の状態や条件に基づいて要素のスタイルを適用するためのCSSの機能です。

/* 擬似クラスの基本例 */
a:hover {
    color: red; /* マウスオーバー時にリンクの色を変更 */
}

擬似要素とは

擬似要素は、要素の特定の部分にスタイルを適用するためのCSSの機能です。例として、要素の最初の行や最初の文字にスタイルを適用できます。

/* 擬似要素の基本例 */
p::first-line {
    font-weight: bold; /* 段落の最初の行を太字にする */
}

:nth-child()擬似クラスの使用

:nth-child()擬似クラスを使用することで、特定の子要素にスタイルを適用できます。

/* nth-child()の例 */
li:nth-child(2n) {
    background-color: lightgray; /* 偶数のリストアイテムに背景色を適用 */
}

:focus擬似クラスの使用

:focus擬似クラスを使用することで、フォーカスされた要素にスタイルを適用できます。

/* focus擬似クラスの例 */
input:focus {
    border: 2px solid blue; /* フォーカス時に青いボーダーを適用 */
}

::beforeおよび::after擬似要素の使用

::beforeおよび::after擬似要素を使用すると、要素の前または後にコンテンツを挿入できます。

/* beforeおよびafterの例 */
h2::before {
    content: "★ "; /* h2要素の前に星マークを追加 */
}
h2::after {
    content: " ★"; /* h2要素の後に星マークを追加 */
}

::selection擬似要素の使用

::selection擬似要素を使うと、選択されたテキストのスタイルを変更できます。

/* selection擬似要素の例 */
::selection {
    background-color: yellow; /* 選択されたテキストの背景色を黄色にする */
}

まとめ

擬似クラスと擬似要素を理解することで、CSSのスタイリングをより柔軟に行うことができ、ユーザー体験を向上させることができます。
