CSSフレックスボックスの理解

フレックスボックスとは

フレックスボックスは、要素を効率的に配置するためのCSSのレイアウトモデルです。親要素にフレックスコンテナを設定し、子要素をフレックスアイテムとして配置します。

/* フレックスボックスの基本設定 */
.container {
    display: flex;
}

フレックスコンテナ

フレックスコンテナは、`display: flex;`を使って設定される要素です。これにより、子要素がフレックスアイテムとして配置されます。

/* フレックスコンテナの例 */
.flex-container {
    display: flex;
    justify-content: center; /* 水平方向の配置 */
}

フレックスアイテムの配置

フレックスアイテムは、フレックスコンテナ内で配置される要素です。`flex-direction`プロパティを使って、配置の方向を指定できます。

/* フレックスアイテムの配置方向 */
.flex-container {
    flex-direction: row; /* 水平方向に配置 */
}

要素のサイズ調整

フレックスアイテムのサイズは、`flex-grow`、`flex-shrink`、`flex-basis`プロパティを使って調整できます。

/* 要素のサイズ調整 */
.flex-item {
    flex-grow: 1; /* 利用可能なスペースを占有 */
    flex-basis: 100px; /* 基本サイズを100pxに設定 */
}

フレックスアイテムの整列

フレックスアイテムの整列は、`align-items`や`justify-content`を使って指定します。これにより、要素の配置を制御できます。

/* フレックスアイテムの整列 */
.flex-container {
    align-items: center; /* 縦方向の中央揃え */
    justify-content: space-between; /* 要素間のスペースを均等に */
}

フレックスラップ

フレックスアイテムがコンテナの幅を超える場合、`flex-wrap`プロパティを使って折り返しを制御できます。

/* フレックスラップの例 */
.flex-container {
    flex-wrap: wrap; /* 要素を折り返す */
}

まとめ

フレックスボックスを理解することで、要素の配置やサイズ調整が容易になり、レスポンシブなデザインを実現できます。
