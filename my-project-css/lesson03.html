CSSボックスモデルの理解

ボックスモデルとは

ボックスモデルは、すべてのHTML要素がボックスとして扱われることを説明します。各ボックスは、コンテンツ、パディング、ボーダー、マージンで構成されています。

/* ボックスモデルの例 */
.box {
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}

コンテンツ領域

ボックスモデルのコンテンツ領域は、実際の内容が表示される部分です。幅と高さは`width`と`height`プロパティで指定します。

/* コンテンツの例 */
.content {
    width: 100px;
    height: 50px;
    background-color: lightblue;
}

パディング

パディングは、ボックスのコンテンツとボーダーの間のスペースを指定します。内側の余白として機能し、`padding`プロパティで設定します。

/* パディングの例 */
.padding-example {
    padding: 15px;
    background-color: lightgreen;
}

ボーダー

ボーダーは、ボックスの周囲を囲む線のことです。ボーダーの幅、スタイル、色を指定することができます。

/* ボーダーの例 */
.border-example {
    border: 2px dashed red;
}

マージン

マージンは、ボックスの外側のスペースを指定します。異なるボックス間の距離を調整するために使用され、`margin`プロパティで設定します。

/* マージンの例 */
.margin-example {
    margin: 20px;
}

ボックスモデルの合計サイズ

ボックスモデルの合計サイズは、コンテンツのサイズにパディング、ボーダー、マージンを加えたものです。これを理解することで、要素のレイアウトがより明確になります。

/* ボックスモデルの合計サイズを計算 */
.total-box {
    width: 220px; /* 幅: 200px + 左右のパディング: 10px + 10px + ボーダー: 2px + 2px */
    height: 120px; /* 高さ: 100px + 上下のパディング: 10px + 10px + ボーダー: 2px + 2px */
}

まとめ

ボックスモデルを理解することで、レイアウトやスタイルの適用をより正確に行うことができ、ウェブページのデザインが向上します。
