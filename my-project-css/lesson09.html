CSSカスタムプロパティの理解

カスタムプロパティとは

カスタムプロパティは、CSSで定義した変数で、スタイルの再利用を可能にします。

/* カスタムプロパティの基本例 */
:root {
    --main-color: blue; /* カスタムプロパティを定義 */
}

.box {
    background-color: var(--main-color); /* カスタムプロパティを使用 */
    width: 100px;
    height: 100px;
}

カスタムプロパティのスコープ

カスタムプロパティは、定義されたスコープ内でのみ有効です。

/* スコープの例 */
:root {
    --primary-bg: lightgray;
}

.container {
    background-color: var(--primary-bg); /* ルートから変数を使用 */
}

.box {
    --box-color: red; /* ボックス固有のカスタムプロパティを定義 */
    background-color: var(--box-color); /* ボックス内で使用 */
}

カスタムプロパティの継承

カスタムプロパティは、親要素から子要素へ継承されます。

/* 継承の例 */
:root {
    --text-color: black;
}

.parent {
    color: var(--text-color); /* 親から色を継承 */
}

.child {
    color: inherit; /* 親の色を引き継ぐ */
}

カスタムプロパティの計算

カスタムプロパティを使って、計算を行うこともできます。

/* 計算の例 */
:root {
    --base-size: 16px;
    --large-size: calc(var(--base-size) * 2); /* 大きさの計算 */
}

.box {
    font-size: var(--large-size); /* 計算結果を使用 */
}

カスタムプロパティのテーマ切替

カスタムプロパティを利用して、テーマの切替を簡単に行うことができます。

/* テーマ切替の例 */
:root {
    --bg-color: white;
    --text-color: black;
}

.dark-theme {
    --bg-color: black;
    --text-color: white;
}

body {
    background-color: var(--bg-color);
    color: var(--text-color);
}

カスタムプロパティのアニメーション

カスタムプロパティは、アニメーションの中でも使用できます。

/* アニメーションにおけるカスタムプロパティ */
@keyframes colorChange {
    0% { --box-color: blue; }
    100% { --box-color: green; }
}

.box {
    animation: colorChange 2s infinite alternate;
    background-color: var(--box-color); /* アニメーション中のカスタムプロパティを使用 */
}

まとめ

カスタムプロパティを使うことで、スタイルの再利用やテーマの切替が簡単になり、CSSの管理がしやすくなります。
