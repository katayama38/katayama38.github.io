グラデーションと影の理解

グラデーションとは

グラデーションは、色が滑らかに変化する背景や要素を作成するためのCSS機能です。

/* 線形グラデーションの例 */
background: linear-gradient(to right, #ff7e5f, #feb47b); /* 左から右へのグラデーション */

放射状グラデーション

放射状グラデーションは、中心から外側に向かって色が変化します。

/* 放射状グラデーションの例 */
background: radial-gradient(circle, #ff7e5f, #feb47b); /* 中心から外側へのグラデーション */

グラデーションの応用

グラデーションをボタンやカードに使用して、視覚的に魅力的なデザインを作成できます。

/* グラデーションを使ったボタン */
button {
    background: linear-gradient(to right, #ff7e5f, #feb47b); /* ボタンのグラデーション */
    color: white; /* テキスト色 */
    padding: 10px 20px; /* 内側の余白 */
    border: none; /* 枠線をなしに */
    border-radius: 5px; /* 角を丸くする */
}

影の基本

影は要素に立体感を与えるためのスタイルで、視覚的に深みを持たせることができます。

/* 影の基本 */
.box {
    width: 300px; /* 幅 */
    height: 200px; /* 高さ */
    background-color: white; /* 背景色 */
    box-shadow: 0 4px 10px rgba(0, 0, 0, 0.1); /* 影のスタイル */
}

影の応用例

影を使って、カードやボタンのデザインを引き立てることができます。

/* 影を使ったカードデザイン */
.card {
    background-color: white; /* カードの背景色 */
    border-radius: 5px; /* 角を丸くする */
    padding: 15px; /* 内側の余白 */
    box-shadow: 0 2px 5px rgba(0, 0, 0, 0.2); /* 影 */
}

複数の影の効果

複数の影を組み合わせることで、より複雑なデザインを作成できます。

/* 複数の影の効果 */
.box {
    width: 300px;
    height: 200px;
    background-color: white;
    box-shadow: 
        0 4px 10px rgba(0, 0, 0, 0.1), /* 軽い影 */
        0 10px 20px rgba(0, 0, 0, 0.2); /* 濃い影 */
}

まとめ

グラデーションと影を活用することで、より魅力的で立体感のあるデザインを作成できます。
