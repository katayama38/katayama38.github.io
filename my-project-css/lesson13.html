CSSメディアクエリの理解

メディアクエリとは

メディアクエリは、異なるデバイスに応じてスタイルを適用するためのCSSの機能です。

/* 基本的なメディアクエリの例 */
@media (max-width: 600px) {
    body {
        background-color: lightblue; /* 小さい画面での背景色 */
    }
}

デバイスの向きに応じたスタイル

デバイスの向き（縦向き、横向き）に応じてスタイルを変更できます。

/* デバイスの向きに応じたスタイル */
@media (orientation: portrait) {
    body {
        font-size: 16px; /* 縦向きの場合のフォントサイズ */
    }
}

@media (orientation: landscape) {
    body {
        font-size: 18px; /* 横向きの場合のフォントサイズ */
    }
}

デバイスの解像度に応じたスタイル

解像度に応じて異なるスタイルを適用できます。

/* 解像度に応じたスタイル */
@media (min-resolution: 192dpi) {
    .high-res {
        background-image: url('high-res-image.jpg'); /* 高解像度画像 */
    }
}

レスポンシブデザインの基本

レスポンシブデザインの基本として、画面サイズに応じてレイアウトを調整します。

/* レスポンシブデザインの基本例 */
@media (max-width: 800px) {
    .container {
        flex-direction: column; /* 小さい画面では縦に並べる */
    }
}

フォントサイズの調整

異なる画面サイズに応じてフォントサイズを調整することが重要です。

/* フォントサイズの調整 */
@media (max-width: 500px) {
    body {
        font-size: 14px; /* 小さい画面用のフォントサイズ */
    }
}

@media (min-width: 501px) {
    body {
        font-size: 16px; /* 大きい画面用のフォントサイズ */
    }
}

カスタムブレークポイントの設定

特定のデザイン要件に応じてカスタムブレークポイントを設定できます。

/* カスタムブレークポイントの例 */
@media (min-width: 1024px) {
    .container {
        max-width: 960px; /* 大きい画面での最大幅 */
        margin: auto; /* 中央揃え */
    }
}

まとめ

メディアクエリを使用することで、デバイスに応じた柔軟なデザインを実現できます。
