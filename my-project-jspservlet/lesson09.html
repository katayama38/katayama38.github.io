セッション管理の理解

セッション管理の概要

セッション管理は、ユーザーの状態を追跡するための手法です。主にログイン状態やカート情報などを管理します。

セッションの作成

新しいセッションを作成するには、request.getSession() メソッドを使用します。

HttpSession session = request.getSession();

セッション属性の設定

セッションに属性を設定するには、setAttribute() メソッドを使用します。

session.setAttribute("username", "ユーザー名");

セッション属性の取得

セッションから属性を取得するには、getAttribute() メソッドを使用します。

String username = (String) session.getAttribute("username");

セッション属性の削除

セッションから属性を削除するには、removeAttribute() メソッドを使用します。

session.removeAttribute("username");

セッションの無効化

セッションを無効化するには、invalidate() メソッドを使用します。

session.invalidate();

セッションの有効期限の設定

セッションの有効期限を設定するには、setMaxInactiveInterval() メソッドを使用します。

session.setMaxInactiveInterval(30 * 60); // 30分

まとめ

セッション管理を適切に行うことで、ユーザーの状態を保持し、より良いユーザー体験を提供できます。
