Java ServletのHTTPリクエストとレスポンスの理解

HTTPリクエストの概要

HTTPリクエストは、クライアントからサーバーに送信されるメッセージで、リソースの取得や操作を要求します。

リクエストメソッド

主なリクエストメソッドには、GET、POST、PUT、DELETEがあります。これらは異なる目的に応じて使用されます。

/* GETリクエストの例 */
protected void doGet(HttpServletRequest request, HttpServletResponse response)
        throws ServletException, IOException {
    // クエリパラメータの取得
    String param = request.getParameter("param");
}

リクエストヘッダー

リクエストヘッダーは、クライアントの情報やリクエストの詳細を含む重要な部分です。

/* リクエストヘッダーの取得例 */
String userAgent = request.getHeader("User-Agent");

HTTPレスポンスの概要

HTTPレスポンスは、サーバーがクライアントに返すメッセージで、リクエストの結果を含みます。

レスポンスの構造

レスポンスは、ステータスライン、ヘッダー、ボディの3つの部分から構成されます。

/* レスポンスの設定例 */
protected void doGet(HttpServletRequest request, HttpServletResponse response)
        throws ServletException, IOException {
    response.setContentType("text/html");
    response.setStatus(HttpServletResponse.SC_OK);
    PrintWriter out = response.getWriter();
    out.println("<h1>レスポンスが送信されました。</h1>");
}
