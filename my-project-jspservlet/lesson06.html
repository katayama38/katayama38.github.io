EL (Expression Language) の使い方の理解

ELの概要

EL (Expression Language) は、JSPで簡潔にオブジェクトのプロパティやコレクションにアクセスするための言語です。

基本的な構文

ELの基本的な構文は、ドル記号と波括弧を使って表現します。例えば、${name}のように記述します。

<%@ page contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%>
<html>
<body>
    <h1>${message}</h1>
</body>
</html>

スコープの理解

ELはリクエストスコープ、セッションスコープ、アプリケーションスコープなど、さまざまなスコープで変数にアクセスできます。

<%@ page contentType="text/html; charset=UTF-8" pageEncoding="UTF-8"%>
<html>
<body>
    <h1>リクエストスコープの変数: ${requestScope.name}</h1>
    <h1>セッションスコープの変数: ${sessionScope.username}</h1>
</body>
</html>

条件文の使用

ELでは条件文を使用して、特定の条件に基づいた出力を制御することができます。

<c:if test="${not empty param.name}">
    <p>こんにちは、${param.name}さん！</p>
</c:if>

コレクションの操作

ELを使用すると、リストやマップなどのコレクションに簡単にアクセスできます。

<%@ taglib uri="http://java.sun.com/jsp/jstl/core" prefix="c" %>
<c:forEach var="item" items="${itemList}">
    <p>アイテム: ${item}</p>
</c:forEach>

ELの演算子

ELでは、比較や論理演算子を使用して条件を評価できます。

<c:if test="${age >= 18}">
    <p>あなたは成人です。</p>
</c:if>

まとめ

ELを理解することで、JSPでのデータ操作がより簡潔になり、コードの可読性が向上します。
