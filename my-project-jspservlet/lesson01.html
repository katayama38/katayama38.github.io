Java Servletの理解

Servletの定義

Servletは、Javaで作られたサーバーサイドのプログラムです。Webアプリケーションのリクエストを処理するために使用されます。

Servletの構成

Servletは主に次の2つのインターフェースを実装します：javax.servlet.Servletとjavax.servlet.http.HttpServlet。

/* 基本的なServletの構成 */
import javax.servlet.*;
import javax.servlet.http.*;
import java.io.*;

public class HelloServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, HttpServletResponse response)
            throws ServletException, IOException {
        response.setContentType("text/html");
        PrintWriter out = response.getWriter();
        out.println("<h1>Hello, Servlet!</h1>");
    }
}

Servletのライフサイクル

Servletは、初期化、リクエスト処理、破棄の3つのフェーズでライフサイクルが管理されます。

/* Servletのライフサイクルメソッド */
public class LifecycleServlet extends HttpServlet {
    public void init() throws ServletException {
        // 初期化処理
    }

    protected void doGet(HttpServletRequest request, HttpServletResponse response)
            throws ServletException, IOException {
        // リクエスト処理
    }

    public void destroy() {
        // 後処理
    }
}

リクエストとレスポンス

Servletは、クライアントからのリクエストを受け取り、適切なレスポンスを返します。

/* リクエストとレスポンスの処理 */
protected void doPost(HttpServletRequest request, HttpServletResponse response)
        throws ServletException, IOException {
    String name = request.getParameter("name");
    response.setContentType("text/html");
    PrintWriter out = response.getWriter();
    out.println("<h1>Hello, " + name + "!</h1>");
}

Web.xmlによる設定

Servletの設定は、web.xmlファイルで行います。


<servlet>
    <servlet-name>HelloServlet</servlet-name>
    <servlet-class>HelloServlet</servlet-class>
</servlet>
<servlet-mapping>
    <servlet-name>HelloServlet</servlet-name>
    <url-pattern>/hello</url-pattern>
</servlet-mapping>

エラーハンドリング

Servletでは、エラーを適切に処理するためのメカニズムが必要です。

/* エラーハンドリング */
public class ErrorHandlingServlet extends HttpServlet {
    protected void doGet(HttpServletRequest request, HttpServletResponse response)
            throws ServletException, IOException {
        try {
            // エラーを発生させる処理
        } catch (Exception e) {
            request.setAttribute("errorMessage", e.getMessage());
            RequestDispatcher dispatcher = request.getRequestDispatcher("/error.jsp");
            dispatcher.forward(request, response);
        }
    }
}
