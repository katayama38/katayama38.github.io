HTML5の新機能: canvas、video、audioの使い方

canvas要素とは

canvas要素は、動的なグラフィックスを描画するためのHTML5の要素です。JavaScriptを使用して、図形や画像を描画できます。

<canvas id="myCanvas" width="200" height="100"></canvas>

JavaScriptを使って、canvasに図形を描画する例:

const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');
ctx.fillStyle = 'red';
ctx.fillRect(0, 0, 200, 100);

video要素の使い方

video要素を使用すると、ウェブページに動画を埋め込むことができます。複数のソース形式を指定することで、互換性を持たせることが可能です。

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    このブラウザは動画に対応していません。
</video>

audio要素の使い方

audio要素を使用すると、音声ファイルをウェブページに埋め込むことができます。これも複数のソース形式を指定することで、さまざまなブラウザでの互換性が向上します。

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    このブラウザは音声に対応していません。
</audio>

まとめ

HTML5のcanvas、video、audio要素を使用することで、よりリッチなインタラクティブコンテンツをウェブページに組み込むことが可能です。
