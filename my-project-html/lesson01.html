初級者っぽい書き方

基本の形を覚えます。

始まりはこれだ！

基本を抑えたコード:

            <!DOCTYPE html>
            <html lang="ja">
            <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Document Title</title>
            </head>
            <body>
            <h1>見出し1</h1>
            <p>これは段落です。</p>
            </body>
            </html>


1, <!DOCTYPE html>:
・HTML5の文書である宣言をします。
2, <html lang="ja">:
・HTML文書の開始を示し、言語の属性を指定します。（ここでは日本語）。
3, <head>:
・文書に関するメタ情報を含む部分です。スタイルやスクリプト、タイトルなどがここに入ります。
4, <meta charset="UTF-8">:
・文書に関するメタ情報を含む部分です。スタイルやスクリプト、タイトルなどがここに含まれます。
5, <meta name="viewport" content="width=device-width, initial-scale=1.0">:
・レスポンシブデザインのための設定です。モバイル端末での表示に適します。
6, <title>:
・ブラウザのタブに表示される文書のタイトルです。
7, <body>:
・実際に表示されるコンテンツを含む部分です。
8, <h1>や<p>:
・見出しや段落などのコンテンツを表すタグです。



私が現在使用しているのは、「Visual Studio Code」です。
「!」エクスクラメーションマークで上記のものは表示できます。使っちゃうよねぇ。。。

中級者っぽい書き方

簡易的な実装コード

        <!DOCTYPE html>
        <html lang="ja">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>中級くらい</title>
        <style>
        body {
            font-family: Arial, sans-serif;
            margin: 20px;
        }
        table {
            border-collapse: collapse;
            width: 100%;
        }
        th, td {
            border: 1px solid #ddd;
            padding: 8px;
        }
        th {
            background-color: #f2f2f2;
        }
        </style>
        </head>
        <body>
        <h1>中級</h1>
        <h2>リスト</h2>
        <p>以下はフルーツのリストです。</p>
        <ul>
            <li>りんご</li>
            <li>バナナ</li>
            <li>オレンジ</li>
        </ul>

        <h2>テーブル</h2>
        <table>
            <tr>
                <th>名前</th>
                <th>年齢</th>
                <th>職業</th>
            </tr>
            <tr>
                <th>佐藤太郎</th>
                <th>30</th>
                <th>エンジニア</th>
            </tr>
            <tr>
                <th>鈴木花子</th>
                <th>25</th>
                <th>デザイナー</th>
            </tr>
        </table>

        <h2>リンクと画像</h2>
        <p>詳しい情報は<a href="https://www.example.com" target="_blank">こちら</a>をクリックしてください。</p>
        <img src="https://via.placeholder.com/150" alt="サンプル画像" />

        <h2>引用</h2>
        <blockquote>
            「知識は力である。」- フランシス・ベーコン
        </blockquote>
        </html>


1, リスト:
・<ul>(順不同リスト)を使用して、フルーツのリストを作成しました。
2, テーブル:
・<table>を使用して、名前、年齢、職業の情報を表形式で表示しました。
3, リンク:
・<a>タグを使用して外部リンクを作成しました。target="_blank"を指定することで新しいタブを開く様になります。
4, 画像:
・<img>タグを使って、プレースホルダー画像を表示しています。
5, 引用:
・<blockquote>を引用文を表示しています。



このコードは中級者向けの要素を含んでおり、基本的なHTMLの使い方に加えて、構造的な要素の使用方法を示しています。

上級者っぽい書き方

フレームワーク、JavaScriptで動的に見せる書き方。

        <!DOCTYPE html>
        <html lang="ja">
        <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <meta name="description" content="上級者っぽい書き方">
        <meta name="keywords" content="HTML, CSS, JavaScript, Bootstrap">
        <title>上級者っぽい書き方</title>
        <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
        <link rel="stylesheet" href="css/style.css">
        <style>
            body {
                padding-top: 56px;
            }
            .footer {
                background-color: #f8f9fa;
                padding: 20px;
                text-align: center;
            }
        </style>
        </head>
        <body>
        <nav class="navbar navbar-expand-lg navbar-light bg-light fixed-top">
            <a class="navbar-brand" href="#">サイトのタイトル</a>
                <button class="navbar-toggler" type="button" data-toggle="collapse" 
                    data-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" 
                    aria-label="Toggler navigation">
                    <span class="navbar-toggler-icon"></span>
                </button>
            <div class="collapse navbar-collapse" id="navbarNav">
                <ul class="navbar-nav">
                    <li class="nav-item active">
                        <a class="nav-link" href="#">ホーム</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#about">私たちについて</a>
                    </li>
                    <li class="nav-item">
                        <a class="nav-link" href="#contact">お問い合わせ</a>
                    </li>
                </ul>
            </div>
        </nav>

        <div class="container">
            <header class="my-5">
                <h1 class="display-4">上級者っぽい書き方</h1>
                <p class="lead">このページは上級者向けな構造を示しています。</p>
            </header>

            <section id="about" class="my-5">
                <h2>私たちについて</h2>
                <p>ここでは、会社やプロジェクトについての詳細を記載します。</p>
                <img src="https://via.placeholder.com/600x400" class="img-fluid" alt="サンプル画像">
            </section>

            <section id="contact" class="my-5">
                <h2>お問い合わせ</h2>
                <form id="contactForm" action="送信先のURL" method="POST">
                    <div class="form-group">
                        <label for="name">名前</label>
                        <input type="text" class="form-control" id="name" required>
                    </div>
                    <div class="form-group">
                        <label for="email">メールアドレス</label>
                        <input type="email" class="form-control" id="email" required>
                    </div>
                    <div class="form-group">
                        <label for="message">メッセージ</label>
                        <textarea class="form-control" id="message" rows="4" required></textarea>
                    </div>
                    <button type="submit" class="btn btn-primary">送信</button>
                </form>
            </section>
        </div>

        <footer class="footer">
            <p>©️ サイト名. All rights reserved.</p>
        </footer>

        <script src="https://code.jquery.com/jquery-3.5.1.slim.min.js"></script>
        <script src="https://cdn.jsdelivr.net/npm/@popperjs/core@2.0.7/dist/umd/popper.min.js"></script>
        <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"></script>
        <script>
            document.getElementById('contactForm').addEventListener('submit', function(event) {
                alert('フォームが送信されました！')
            });
        </script>
        </body>
        </html>

・固定型のナビゲーションバーを実装し、ページ内リンクを設定しています。
3, セクションの分割:
・ページのセクションごとに分けて、内容を明確にしています。
4, フォーム:
・ユーザーからの情報を取得するためのフォームを作成し、JavaScriptで送信イベントを処理しています。
5, スタイルのカスタマイズ:
・CSSで簡単なスタイルを追加し、ページの見た目を整えています。

この様に、上級者向けのHTMLでは、実用的な要素やフレームワークを組み合わせて、より機能的で視覚的に魅力的なページを作成しています。

まとめ

HTMLの基本構造と要素

HTML(HyperText Markup Language)は、ウェブページを構築するための基本的な言語です。HTML文書は、主に以下の部分から構成されています。

1, 初級まとめ: 基本的な構造

<!DOCTYPE html>:
・HTML5の文書である宣言。これは骨組み、骨格です!みたいなこと。

<html lang="ja">
・文書の言語を指定。"en"は英語のWebサイトですよ。とは"ja"はJapanの略である。

<head>部分
・ページのメタ情報。
・例えば<title>ページのタイトル</title><meta charset="UTF-8">など。
・重要なポイント: この内容は通常時ユーザーには表示されない。右クリック「検証」で見ることができたりする。

<body>部分:
・実際に表示されるコンテンツが含まれる。
・例えば: <h1>(見出し) <p>(段落) <img>(画像) などがある。


2, 中級まとめ: セマンティック要素とスタイリング

・セマンティック要素:
<header>, <nav>, <main>, <footer>などの要素を使って文書を理論的に構造化。現実的にすると「机の上」とか「テーブル上の配置」とか。
これにより、検索エンジンや支援技術にとって情報が理解しやすくなると。「ねぇねぇ！あれとって！」ができやすくなるイメージ。

・スタイルの適用:
CSSを使用して、ウェブページの見た目を整える。CSS。これが奥が深いのと、まるで2000ピースパズルをやっているみたいで、楽しい。
外部スタイルシートをリンクさせるのが一般的。<link rel="stylesheet" href="style.css>/, や ./, や ../なども後々理解。

・フォーム:
・ユーザーからの情報を収集するためのインタラクティブな要素を追加。インタラクティブ？

3, 上級まとめ: インタラクティブな要素とフレームワークの活用

・JavaScriptの統合:
・ページの動的な要素を作成するためにJavaScriptを使用します。フォームの送信やユーザーインタラクションに応じた処理を行います。
