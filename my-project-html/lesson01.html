HTML文書構造の理解

基本構造

HTML文書は、必ず<!DOCTYPE>宣言から始まり、その後に<html>タグが続きます。<head>タグには文書のメタ情報が含まれ、<body>タグには実際のコンテンツが含まれます。

<!DOCTYPE html>
<html lang="ja">
    <head>
        <meta charset="UTF-8">
        <title>タイトル</title>
    </head>
    <body>
        <!-- コンテンツ -->
    </body>
</html>

要素の役割

HTML要素は、テキスト、画像、リンクなどのコンテンツを構成する基本的な部分です。各要素には役割があり、正しい要素を使うことで文書の意味が明確になります。

<p>これは段落です。</p>
<a href="https://example.com">リンク</a>

セクション要素

セクション要素を使用することで、文書を論理的に構造化できます。<header>、<footer>、<nav>、<section>などの要素があります。

<header>
    <h1>ウェブサイトのタイトル</h1>
</header>

<nav>
    <a href="#section1">セクション1</a>
</nav>

<section>
    <h2>セクション見出し</h2>
    <p>セクションの内容です。</p>
</section>

メタ情報の重要性

メタ情報は、文書のSEOやアクセシビリティに影響を与えます。例えば、<meta>タグを使用して、文字エンコーディングやページの説明を指定します。

<head>
    <meta charset="UTF-8">
    <meta name="description" content="ページの説明">
    <meta name="keywords" content="キーワード1, キーワード2">
</head>

まとめ

HTML文書の構造を理解することで、より意味のあるウェブページを作成することができます。
