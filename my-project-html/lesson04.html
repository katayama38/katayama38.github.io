HTMLリストの作成方法

順不同リスト

順不同リストは、<ul>タグを使用して作成します。各項目は<li>タグで囲みます。

<ul>
    <li>りんご</li>
    <li>バナナ</li>
    <li>オレンジ</li>
</ul>

順序付きリスト

順序付きリストは、<ol>タグを使用して作成します。こちらも項目は<li>タグで囲みます。

<ol>
    <li>第一位</li>
    <li>第二位</li>
    <li>第三位</li>
</ol>

入れ子リスト

リストは入れ子にすることもできます。<ul>や<ol>の中に別のリストを含めることができます。

<ul>
    <li>果物
        <ul>
            <li>りんご</li>
            <li>バナナ</li>
        </ul>
    </li>
    <li>野菜
        <ol>
            <li>人参</li>
            <li>トマト</li>
        </ol>
    </li>
</ul>

まとめ

リストを正しく使うことで、情報を整理して視覚的にわかりやすく表示できます。
