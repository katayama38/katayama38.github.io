75. ローカルストレージの活用

Vue.jsを使って、ブラウザのローカルストレージにデータを保存し、アプリケーションが再読み込みされたときにもデータを保持する方法を紹介します。

<!-- ローカルストレージを利用してデータを保存 -->
<template>
  <div>
    <input v-model="userData" @input="saveData" placeholder="ユーザー情報を入力">
    <p>保存された情報: {{ savedData }}</p>
  </div>
</template>

<script>
export default {
  data() {
    return {
      userData: '',
      savedData: localStorage.getItem('userData') || ''
    };
  },
  methods: {
    saveData() {
      localStorage.setItem('userData', this.userData);
      this.savedData = this.userData;
    }
  }
};
</script>

76. ダッシュボードの構築

Vue.jsを使って、リアルタイムのデータを視覚的に表示するダッシュボードを構築する方法を紹介します。`chart.js`や`vue-chartjs`ライブラリを使用して、グラフやチャートを表示します。

<!-- ダッシュボード用のチャート表示 -->
<template>
  <div>
    <h2>データダッシュボード</h2>
    <line-chart :data="chartData"></line-chart>
  </div>
</template>

<script>
import { Line } from 'vue-chartjs';
import { reactive } from 'vue';
import { Chart as ChartJS } from 'chart.js';

export default {
  components: {
    LineChart: Line
  },
  data() {
    return {
      chartData: reactive({
        labels: ['1月', '2月', '3月', '4月'],
        datasets: [
          {
            label: '売上',
            data: [100, 200, 300, 400],
            borderColor: 'rgb(75, 192, 192)',
            fill: false
          }
        ]
      })
    };
  }
};
</script>

77. ユーザーインターフェースのテスト

Vue.jsのコンポーネントに対して、Vue Test Utilsを使ってユニットテストを実施する方法を紹介します。テストを通じて、UIコンポーネントが期待通りに動作することを確認できます。

<!-- コンポーネントのユニットテスト -->
<template>
  <button @click="increment">{{ count }}</button>
</template>

<script>
export default {
  data() {
    return {
      count: 0
    };
  },
  methods: {
    increment() {
      this.count++;
    }
  }
};
</script>

<!-- テストコード -->
import { mount } from '@vue/test-utils';
import Counter from './Counter.vue';

test('ボタンがクリックされるとカウントが増える', async () => {
  const wrapper = mount(Counter);
  await wrapper.find('button').trigger('click');
  expect(wrapper.text()).toContain('1');
});

79. プロジェクトの国際化（i18n）

Vue.jsで多言語対応を実現するため、`vue-i18n`ライブラリを使った国際化の設定方法を紹介します。これにより、アプリケーションを複数の言語に対応させることができます。

<!-- vue-i18nを使った国際化 -->
<template>
  <div>
    <p>{{ $t("message") }}</p>
  </div>
</template>

<script>
import { createI18n } from 'vue-i18n';

const messages = {
  en: { message: 'Hello!' },
  ja: { message: 'こんにちは!' }
};

const i18n = createI18n({
  locale: 'ja',
  messages
});

export default {
  i18n
};
</script>

80. RESTful APIとの統合

Vue.jsアプリケーションでRESTful APIを利用して、データを取得する方法を紹介します。Axiosを使用して非同期にデータを取得し、アプリケーションに表示します。

<!-- RESTful APIからデータを取得 -->
<template>
  <div>
    <ul>
      <li v-for="item in items" :key="item.id">{{ item.name }}</li>
    </ul>
  </div>
</template>

<script>
import axios from 'axios';

export default {
  data() {
    return {
      items: []
    };
  },
  async mounted() {
    const response = await axios.get('https://api.example.com/items');
    this.items = response.data;
  }
};
</script>

83. ユーザー設定の管理

アプリケーション内でユーザーの設定を保存し、次回起動時にその設定を適用する方法を紹介します。ローカルストレージやVuexを使ってユーザー設定を管理できます。

<!-- ユーザー設定を保存・管理 -->
<template>
  <div>
    <label for="theme">テーマ:</label>
    <select v-model="theme" @change="saveSettings">
      <option value="light">ライト</option>
      <option value="dark">ダーク</option>
    </select>
  </div>
</template>

<script>
export default {
  data() {
    return {
      theme: localStorage.getItem('theme') || 'light'
    };
  },
  methods: {
    saveSettings() {
      localStorage.setItem('theme', this.theme);
    }
  }
};
</script>
