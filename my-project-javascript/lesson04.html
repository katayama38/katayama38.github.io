JavaScriptの関数の理解

関数の定義

関数は、一連の処理をまとめたもので、再利用可能なコードのブロックです。

/* 関数の定義の例 */
function greet() {
    console.log("こんにちは！");
}
greet(); // "こんにちは！"と表示される

引数を持つ関数

関数は引数を受け取り、処理に利用することができます。

/* 引数を持つ関数の例 */
function add(a, b) {
    return a + b;
}
let sum = add(3, 4); // sumは7になる
console.log(sum);

戻り値のある関数

関数は、処理の結果を戻り値として返すことができます。

/* 戻り値のある関数の例 */
function square(x) {
    return x * x;
}
console.log(square(5)); // 25と表示される

無名関数

無名関数は名前を持たない関数で、変数に代入して使用されます。

/* 無名関数の例 */
const multiply = function(a, b) {
    return a * b;
};
console.log(multiply(2, 3)); // 6と表示される

アロー関数

アロー関数は、より短い構文で関数を定義する方法です。

/* アロー関数の例 */
const divide = (a, b) => a / b;
console.log(divide(10, 2)); // 5と表示される

関数のスコープ

関数内で定義された変数は、その関数のスコープ内でのみ有効です。

/* 関数のスコープの例 */
function scopeTest() {
    let localVar = "私はローカル変数です";
    console.log(localVar);
}
scopeTest(); // "私はローカル変数です"と表示される
// console.log(localVar); // エラー：localVarは定義されていません

まとめ

JavaScriptの関数を理解することで、再利用可能なコードを作成し、プログラムの構造をより整理できます。
