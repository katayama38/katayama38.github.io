カウントダウンタイマーの理解

カウントダウンタイマーの基本

このカウントダウンタイマーは、指定した秒数からカウントダウンし、時間が切れるとメッセージを表示します。

カウントダウン結果の表示

カウントダウンの結果はボタンをクリックした後、下部に表示されます。

エラーハンドリング

無効な入力があった場合、エラーメッセージを表示します。正しい数値を入力しないとカウントダウンは始まりません。

JavaScript コードの説明

以下のコードは、カウントダウンタイマーの機能を実現しています：

document.getElementById('start').addEventListener('click', function() {
    let seconds = parseInt(document.getElementById('seconds').value);
    const timerDisplay = document.getElementById('timer');

    const interval = setInterval(function() {
        if (seconds <= 0) {
            clearInterval(interval);
            timerDisplay.innerText = '時間切れ！';
        } else {
            timerDisplay.innerText = seconds + '秒';
            seconds--;
        }
    }, 1000);
});

このコードの説明は以下の通りです：

  • イベントリスナーの設定: スタートボタンがクリックされたときに実行される関数を設定します。
  • 秒数の取得: ユーザーが入力した秒数を取得し、整数に変換します。
  • カウントダウンの処理: setIntervalを使って1秒ごとに秒数を減らし、表示を更新します。
  • 時間切れの処理: 秒数が0以下になるとタイマーを停止し、「時間切れ！」というメッセージを表示します。

まとめ

このカウントダウンタイマーを使って、簡単に時間を管理することができます。正しい秒数を入力することで、正確なカウントダウンが行えます。
