JavaScriptの制御フローの理解

条件分岐

条件分岐は、特定の条件に基づいて異なるコードを実行するために使用されます。

/* if文の例 */
let score = 85;
if (score >= 60) {
    console.log("合格");
} else {
    console.log("不合格");

switch文

switch文は、変数の値に応じて異なる処理を行うための構文です。

/* switch文の例 */
let fruit = "りんご";
switch (fruit) {
    case "バナナ":
        console.log("バナナが好きです");
        break;
    case "りんご":
        console.log("りんごが好きです");
        break;
    default:
        console.log("他の果物が好きです");
}

ループ処理

ループは、指定した回数または条件が満たされるまで繰り返し処理を行うための構文です。

/* for文の例 */
for (let i = 0; i < 5; i++) {
    console.log("カウント: " + i);
}

while文

while文は、指定した条件が真である限り、処理を繰り返すために使用されます。

/* while文の例 */
let count = 0;
while (count < 5) {
    console.log("カウント: " + count);
    count++;
}

do...while文

do...while文は、少なくとも一度は処理を実行した後に条件を確認します。

/* do...while文の例 */
let number = 0;
do {
    console.log("カウント: " + number);
    number++;
} while (number < 5);

breakとcontinue

breakはループを終了し、continueは次のイテレーションにスキップします。

/* breakとcontinueの例 */
for (let i = 0; i < 10; i++) {
    if (i === 5) {
        break; // 5に達したらループを終了
    }
    console.log(i);
}

まとめ

JavaScriptの制御フローを理解することで、プログラムの動作を柔軟に制御することができ、より複雑なロジックを実装できます。
