JavaScriptの非同期処理の理解

非同期処理の定義

非同期処理とは、処理が完了するのを待たずに次の処理を実行できるプログラミングの手法です。

/* 非同期処理の定義の例 */
console.log("開始");
setTimeout(() => {
    console.log("非同期処理完了");
}, 2000);
console.log("終了");

コールバック関数

非同期処理が完了したときに実行される関数をコールバック関数と呼びます。

/* コールバック関数の例 */
function fetchData(callback) {
    setTimeout(() => {
        callback("データを取得しました");
    }, 1000);
}

fetchData(data => {
    console.log(data);
});

Promiseの基本

Promiseは、非同期処理の結果を表すオブジェクトです。

/* Promiseの基本の例 */
let myPromise = new Promise((resolve, reject) => {
    setTimeout(() => {
        resolve("Promiseが解決されました");
    }, 1500);
});

myPromise.then(result => {
    console.log(result);
});

非同期関数とawait

非同期関数を使うことで、Promiseの結果を簡潔に処理できます。

/* 非同期関数とawaitの例 */
async function asyncFunction() {
    let result = await myPromise;
    console.log(result);
}

asyncFunction();

複数の非同期処理の実行

Promise.allを使うことで、複数の非同期処理を同時に待つことができます。

/* 複数の非同期処理の実行の例 */
let promise1 = new Promise(resolve => setTimeout(() => resolve("結果1"), 1000));
let promise2 = new Promise(resolve => setTimeout(() => resolve("結果2"), 2000));

Promise.all([promise1, promise2]).then(results => {
    console.log(results); // ["結果1", "結果2"]
});

まとめ

JavaScriptの非同期処理を理解することで、効率的に時間のかかる処理を扱えるようになります。
