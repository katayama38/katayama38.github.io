JavaScriptの高階関数の理解

高階関数の定義

高階関数とは、他の関数を引数に取ったり、関数を返したりする関数のことです。

/* 高階関数の定義の例 */
function highOrderFunction(fn) {
    return function(x) {
        return fn(x);
    };
}

関数を引数に取る高階関数

高階関数に関数を渡すことで、動的に処理を変更できます。

/* 関数を引数に取る高階関数の例 */
function greet(name) {
    return `こんにちは、${name}さん！`;
}

function processGreeting(fn, name) {
    return fn(name);
}

console.log(processGreeting(greet, "田中")); // こんにちは、田中さん！

関数を返す高階関数

関数を返す高階関数を使って、カスタム関数を作成できます。

/* 関数を返す高階関数の例 */
function createMultiplier(multiplier) {
    return function(x) {
        return x * multiplier;
    };
}

const double = createMultiplier(2);
console.log(double(5)); // 10

Arrayメソッドと高階関数

map、filter、reduceなどの配列メソッドは高階関数です。

/* Arrayメソッドと高階関数の例 */
const numbers = [1, 2, 3, 4, 5];

const squares = numbers.map(x => x * x);
console.log(squares); // [1, 4, 9, 16, 25]

コールバック関数としての高階関数

高階関数はコールバック関数を受け取ることができます。

/* コールバック関数としての高階関数の例 */
function processArray(arr, callback) {
    return arr.map(callback);
}

const doubled = processArray([1, 2, 3], x => x * 2);
console.log(doubled); // [2, 4, 6]

高階関数の利点

高階関数を使用することで、コードの再利用性や抽象化が向上します。

/* 高階関数の利点の例 */
console.log("高階関数を利用した再利用性の向上");

まとめ

JavaScriptの高階関数を理解することで、柔軟で保守性の高いコードを作成できます。
