JavaScriptの基礎の理解

JavaScriptの基本構造

JavaScriptは、ウェブページに動的な機能を追加するためのプログラミング言語です。基本的な文法を理解することが重要です。

/* 基本的な変数宣言 */
let message = "こんにちは、世界！";
console.log(message);

データ型

JavaScriptには、数値、文字列、配列、オブジェクトなど、いくつかの基本的なデータ型があります。

/* データ型の例 */
let number = 42; // 数値
let text = "文字列"; // 文字列
let array = [1, 2, 3]; // 配列
let object = { key: "value" }; // オブジェクト

条件分岐

条件分岐は、特定の条件に応じて異なるコードを実行するために使用されます。

/* 条件分岐の例 */
let score = 85;
if (score >= 60) {
    console.log("合格");
} else {
    console.log("不合格");
}

ループ処理

ループは、指定した回数または条件が満たされるまで繰り返し処理を行うための構文です。

/* ループ処理の例 */
for (let i = 0; i < 5; i++) {
    console.log("カウント: " + i);
}

関数

関数は、一連の処理をまとめて再利用可能にするための構文です。

/* 関数の例 */
function greet(name) {
    return "こんにちは、" + name + "!";
}
console.log(greet("太郎"));

イベント処理

JavaScriptでは、ユーザーのアクションに応じて処理を実行するためのイベントを扱うことができます。

/* イベント処理の例 */
document.getElementById("myButton").addEventListener("click", function() {
    alert("ボタンがクリックされました！");
});

まとめ

JavaScriptを理解することで、ウェブページにインタラクティブな要素を追加し、ユーザー体験を向上させることができます。
