JavaScriptのDOMイベントの理解

DOMイベントの定義

DOMイベントは、ユーザーがブラウザで行ったアクションに応じて発生するものです。

/* DOMイベントの定義の例 */
document.addEventListener('click', function() {
    console.log('クリックされました！');
});

イベントリスナーの追加

特定の要素にイベントリスナーを追加することで、イベントを監視することができます。

/* 要素にイベントリスナーを追加する例 */
const button = document.getElementById('myButton');
button.addEventListener('click', function() {
    alert('ボタンがクリックされました！');
});

イベントオブジェクトの利用

イベントハンドラーの引数には、イベントオブジェクトが渡されます。

/* イベントオブジェクトの利用の例 */
button.addEventListener('click', function(event) {
    console.log(event); // イベントオブジェクトを表示
});

イベントの伝播

イベントは親要素に向かって伝播します（バブリング）。これを利用して、親要素でイベントを処理することができます。

/* イベントの伝播の例 */
const parentDiv = document.getElementById('parent');
parentDiv.addEventListener('click', function() {
    alert('親要素がクリックされました！');
});

デフォルトの動作の防止

特定のイベントのデフォルトの動作を防ぐには、`event.preventDefault()`を使用します。

/* デフォルトの動作を防ぐ例 */
const link = document.getElementById('myLink');
link.addEventListener('click', function(event) {
    event.preventDefault();
    console.log('リンクのデフォルト動作を防ぎました。');
});

複数のイベントリスナー

同じ要素に複数のイベントリスナーを追加することも可能です。

/* 複数のイベントリスナーの例 */
button.addEventListener('click', function() {
    console.log('最初のリスナー');
});
button.addEventListener('click', function() {
    console.log('次のリスナー');
});

まとめ

JavaScriptのDOMイベントを理解することで、ユーザーインターフェースをよりインタラクティブにすることができます。
