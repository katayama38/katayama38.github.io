JavaScriptのDOM操作の理解

DOMの定義

DOM（Document Object Model）は、HTMLやXML文書をプログラムから操作するためのインターフェースです。

/* DOMの定義の例 */
console.log(document); // 現在の文書のDOMオブジェクトを表示

要素の取得

DOMの要素を取得する方法は複数あります。

/* 要素の取得の例 */
let header = document.getElementById('header');
let paragraphs = document.getElementsByTagName('p');
let links = document.querySelectorAll('a');
console.log(header, paragraphs, links);

要素の変更

取得した要素の内容や属性を変更することができます。

/* 要素の変更の例 */
header.textContent = "新しいヘッダー"; // テキスト内容を変更
header.setAttribute("class", "new-class"); // クラス属性を変更
console.log(header);

要素の追加

新しい要素を作成してDOMに追加することができます。

/* 要素の追加の例 */
let newElement = document.createElement('div');
newElement.textContent = "新しい要素です";
document.body.appendChild(newElement); // bodyに追加
console.log(newElement);

要素の削除

DOMから要素を削除することも可能です。

/* 要素の削除の例 */
let elementToRemove = document.getElementById('removeMe');
elementToRemove.parentNode.removeChild(elementToRemove); // 要素を削除
console.log("要素が削除されました");

イベントの追加

DOM要素に対してイベントリスナーを追加できます。

/* イベントの追加の例 */
header.addEventListener('click', function() {
    alert("ヘッダーがクリックされました");
});

まとめ

JavaScriptのDOM操作を理解することで、ウェブページの内容を動的に変更できるようになります。
