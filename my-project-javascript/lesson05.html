JavaScriptの配列の理解

配列の定義

配列は、複数の値を格納できるデータ構造です。

/* 配列の定義の例 */
let fruits = ["りんご", "バナナ", "オレンジ"];
console.log(fruits); // ["りんご", "バナナ", "オレンジ"]

配列の要素へのアクセス

配列の要素にはインデックスを使ってアクセスします。

/* 配列の要素へのアクセスの例 */
let firstFruit = fruits[0]; // "りんご"
console.log(firstFruit);

配列の長さ

配列の要素数は、lengthプロパティを使って取得できます。

/* 配列の長さの例 */
console.log(fruits.length); // 3と表示される

配列の追加と削除

push()メソッドで要素を追加し、pop()メソッドで要素を削除できます。

/* 配列の追加と削除の例 */
fruits.push("ぶどう"); // "ぶどう"を追加
console.log(fruits); // ["りんご", "バナナ", "オレンジ", "ぶどう"]
let lastFruit = fruits.pop(); // 最後の要素を削除
console.log(lastFruit); // "ぶどう"
console.log(fruits); // ["りんご", "バナナ", "オレンジ"]

配列のループ処理

forEach()メソッドを使って配列の各要素を処理できます。

/* 配列のループ処理の例 */
fruits.forEach(function(fruit) {
    console.log(fruit);
});

配列のスライス

slice()メソッドを使って配列の一部を新しい配列として取得できます。

/* 配列のスライスの例 */
let citrus = fruits.slice(1, 3); // ["バナナ", "オレンジ"]
console.log(citrus);

まとめ

JavaScriptの配列を理解することで、複数のデータを効率的に管理できるようになります。
