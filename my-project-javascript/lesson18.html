JavaScriptのWeb APIの理解

Web APIの定義

Web APIは、ウェブアプリケーションが外部のサービスとやり取りするためのインターフェースです。

/* Web APIの使用例 */
fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data));

Fetch API

Fetch APIは、ネットワークリクエストを簡単に行うためのインターフェースです。

/* Fetch APIの基本的な使い方 */
fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data));

Geolocation API

Geolocation APIを使用すると、ユーザーの地理的位置を取得できます。

/* Geolocation APIの使用例 */
navigator.geolocation.getCurrentPosition(position => {
    console.log(position.coords.latitude, position.coords.longitude);
});

Local Storage API

Local Storageを使用すると、ブラウザにデータを永続的に保存できます。

/* Local Storage APIの使用例 */
localStorage.setItem('key', 'value');
const value = localStorage.getItem('key');
console.log(value); // 'value'

WebSockets API

WebSocketsを使用すると、サーバーとクライアント間で双方向通信が可能になります。

/* WebSockets APIの使用例 */
const socket = new WebSocket('wss://example.com/socket');
socket.onmessage = function(event) {
    console.log(event.data);
};

Notification API

Notification APIを使用すると、ウェブアプリからデスクトップ通知を表示できます。

/* Notification APIの使用例 */
if (Notification.permission === 'granted') {
    new Notification('タイトル', { body: '通知の内容' });
} else {
    Notification.requestPermission();
}

まとめ

JavaScriptのWeb APIを理解することで、よりリッチなウェブアプリケーションを構築することができます。
