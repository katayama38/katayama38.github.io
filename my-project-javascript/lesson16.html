JavaScriptのJSONの理解

JSONの定義

JSON（JavaScript Object Notation）は、データを軽量で簡潔に表現するためのフォーマットです。

/* JSONの定義の例 */
const person = {
    "name": "太郎",
    "age": 25,
    "isStudent": false
};

JSONの文字列化

JavaScriptオブジェクトをJSON文字列に変換するには、`JSON.stringify`メソッドを使用します。

/* JSONの文字列化の例 */
const jsonString = JSON.stringify(person);
console.log(jsonString); // {"name":"太郎","age":25,"isStudent":false}

JSONの解析

JSON文字列をJavaScriptオブジェクトに変換するには、`JSON.parse`メソッドを使用します。

/* JSONの解析の例 */
const parsedPerson = JSON.parse(jsonString);
console.log(parsedPerson.name); // 太郎

JSONの配列

JSONでは配列も定義できます。複数のオブジェクトを格納することが可能です。

/* JSONの配列の例 */
const people = [
    { "name": "太郎", "age": 25 },
    { "name": "次郎", "age": 30 }
];

JSONとAJAX

AJAXを使用してサーバーからJSONデータを取得することができます。

/* AJAXでJSONを取得する例 */
fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data));

JSONの注意点

JSONではプロパティ名はダブルクォーテーションで囲む必要があります。また、関数やundefinedはサポートされていません。

/* JSONの注意点 */
const invalidJson = {
    name: "太郎",
    age: undefined // これは無効
};

まとめ

JavaScriptのJSONを理解することで、データの送受信や保存が容易になり、効率的なプログラミングが可能になります。
