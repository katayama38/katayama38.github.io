計算機の基本

この計算機は、2つの数を入力し、その合計を計算します。ボタンをクリックすると、計算結果が表示されます。

エラーハンドリング

数値以外の入力があった場合、エラーメッセージを表示します。無効な入力がある場合は、計算が行われず、ユーザーに注意を促します。

JavaScript コードの説明

以下のコードは、計算機の機能を実現しています：

document.addEventListener('DOMContentLoaded', function() {
    const calculateButton = document.getElementById('calculate');
    if (calculateButton) {
        calculateButton.addEventListener('click', function() {
            const num1 = parseFloat(document.getElementById('num1').value);
            const num2 = parseFloat(document.getElementById('num2').value);

            if (isNaN(num1) || isNaN(num2)) {
                document.getElementById('calcResult').innerText = '有効な数を入力してください。';
                return;
            }

            const sum = num1 + num2;
            document.getElementById('calcResult').innerText = '合計: ' + sum;
        });
    } else {
        console.error("ID 'calculate' の要素が見つかりません。");
    }
});

このコードの説明は以下の通りです：

  • DOMContentLoaded イベントリスナー: DOMが完全に読み込まれた後に実行される関数を設定します。
  • ボタンの取得とイベントリスナーの設定: 計算ボタンを取得し、クリック時の処理を追加します。
  • 数値の取得と処理: ユーザーが入力した数値を取得し、数値型に変換します。
  • エラーチェック: 入力が有効であるかを確認し、無効な場合はエラーメッセージを表示します。
  • 計算と結果の表示: 入力された2つの数値を加算し、結果を表示します。

まとめ

この計算機を使って、簡単な計算を行うことができます。正しい数値を入力することで、正確な結果が得られます。
