JavaScriptの演算子の理解

算術演算子

算術演算子は、数値の計算を行うために使用されます。

/* 算術演算子の例 */
let a = 10;
let b = 5;
let sum = a + b; // 加算
let difference = a - b; // 減算
let product = a * b; // 乗算
let quotient = a / b; // 除算

代入演算子

代入演算子は、変数に値を代入するために使用されます。

/* 代入演算子の例 */
let x = 10; // 代入
x += 5; // x = x + 5
x -= 3; // x = x - 3

比較演算子

比較演算子は、2つの値を比較して真または偽を返します。

/* 比較演算子の例 */
let c = 10;
let d = 20;
console.log(c > d); // false
console.log(c < d); // true
console.log(c === 10); // true

論理演算子

論理演算子は、論理値を操作するために使用されます。

/* 論理演算子の例 */
let e = true;
let f = false;
console.log(e && f); // false (AND)
console.log(e || f); // true (OR)
console.log(!e); // false (NOT)

ビット演算子

ビット演算子は、整数をビット単位で操作するために使用されます。

/* ビット演算子の例 */
let g = 5; // 0101
let h = 3; // 0011
console.log(g & h); // 1 (AND)
console.log(g | h); // 7 (OR)
console.log(g ^ h); // 6 (XOR)

条件演算子（三項演算子）

条件演算子は、条件に応じて異なる値を返すために使用されます。

/* 条件演算子の例 */
let age = 18;
let canVote = (age >= 18) ? "投票できます" : "投票できません";
console.log(canVote);

まとめ

JavaScriptの演算子を理解することで、さまざまな計算や条件判断が可能になり、プログラムの柔軟性が向上します。
