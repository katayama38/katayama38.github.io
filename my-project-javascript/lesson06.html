JavaScriptのオブジェクトの理解

オブジェクトの定義

オブジェクトは、プロパティとメソッドを持つデータ構造です。

/* オブジェクトの定義の例 */
let car = {
    brand: "トヨタ",
    model: "カムリ",
    year: 2021
};
console.log(car); // {brand: "トヨタ", model: "カムリ", year: 2021}

プロパティへのアクセス

オブジェクトのプロパティにはドット記法またはブラケット記法でアクセスできます。

/* プロパティへのアクセスの例 */
let carBrand = car.brand; // "トヨタ"
let carModel = car["model"]; // "カムリ"
console.log(carBrand, carModel);

プロパティの追加と削除

新しいプロパティを追加したり、既存のプロパティを削除できます。

/* プロパティの追加と削除の例 */
car.color = "赤"; // 新しいプロパティを追加
delete car.year; // プロパティを削除
console.log(car); // {brand: "トヨタ", model: "カムリ", color: "赤"}

メソッドの定義

オブジェクト内に関数を定義して、メソッドとして使用できます。

/* メソッドの定義の例 */
car.getDetails = function() {
    return `${this.brand} ${this.model} (${this.color})`;
};
console.log(car.getDetails()); // "トヨタ カムリ (赤)"

オブジェクトのループ処理

for...in文を使ってオブジェクトのプロパティをループ処理できます。

/* オブジェクトのループ処理の例 */
for (let key in car) {
    console.log(key + ": " + car[key]);
}

ネストされたオブジェクト

オブジェクトの中に他のオブジェクトを含めることができます。

/* ネストされたオブジェクトの例 */
let person = {
    name: "山田太郎",
    address: {
        city: "東京",
        zip: "100-0001"
    }
};
console.log(person.address.city); // "東京"

まとめ

JavaScriptのオブジェクトを理解することで、データをより構造化して管理できるようになります。
